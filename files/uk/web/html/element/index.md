{{HTMLSidebar("HTML_Elements")}}

Ця сторінка перелічує всі {{Glossary("Element","елементи")}} {{Glossary("HTML")}}, котрі можна створити за допомогою {{Glossary("Tag", "тегів")}}.

Вони згруповані за функціями, щоб допомогти структурувати їх розуміння. Алфавітний список усіх елементів доступний у боковій панелі на сторінці кожного елемента, як і на цій.

> **Примітка:** Для отримання докладної інформації про основи елементів та атрибутів HTML дивіться [розділ про елементи у статті Вступ до HTML](/uk/docs/Learn/HTML/Introduction_to_HTML#elementy-osnovni-budmaterialy).

## Головний корінь

| Елемент                 | Опис                                                                                                                                                               |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("html")}} | Представляє корінь (елемент найвищого рівня) документа HTML, тож його також звуть _кореневим елементом_. Всі решта елементів мусять бути нащадками цього елемента. |

## Метадані документа

Метадані містять інформацію про сторінку. Вона включає інформацію про стилі, сценарії та дані, що допомагають програмам ({{Glossary("search engine", "пошуковим системам")}}, {{Glossary("Browser","браузерам")}} тощо) у використанні та візуалізації сторінки. Метадані стилів та сценаріїв можуть бути визначені у сторінці або зв'язані з іншим файлом, що містить інформацію.

| Елемент                  | Опис                                                                                                                                                                                                                                                                                                         |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("base")}}  | Задає базовий URL для використання всіма відносними URL у документі. В документі може бути лише один такий елемент.                                                                                                                                                                                          |
| {{HTMLElement("head")}}  | Містить машинно-читабельну інформацію (метадані) про документ, як то його [назву](/uk/docs/Web/HTML/Element/title), [сценарії](/uk/docs/Web/HTML/Element/script) та [списки стилів](/uk/docs/Web/HTML/Element/style).                                                                                        |
| {{HTMLElement("link")}}  | Задає відносини між поточним документом та зовнішнім ресурсом. Цей елемент найчастіше використовується для посилань на CSS, але також вживається, окрім іншого, для встановлення піктограми сайту (і піктограм у стилі "favicon", так і піктограми для робочого столу й застосунків на мобільних пристроях). |
| {{HTMLElement("meta")}}  | Представляє {{Glossary("Metadata","метадані")}}, котрі не можуть бути представлені в інших елементах HTML для метаданих, як то {{HTMLElement("base")}}, {{HTMLElement("link")}}, {{HTMLElement("script")}}, {{HTMLElement("style")}} і {{HTMLElement("title")}}.                                             |
| {{HTMLElement("style")}} | Містить інформацію про стилі для документа або його частини. Містить CSS, котрий застосовується до вмісту документа, що містить цей елемент.                                                                                                                                                                 |
| {{HTMLElement("title")}} | Задає назву документа, що виводиться в смузі заголовка {{glossary("Browser", "браузера")}} або вкладки сторінки. Містить лише текст, теги всередині такого елемента – ігноруються.                                                                                                                           |

## Розділення вмісту

Елементи розділення вмісту дають змогу впорядкувати вміст документа у логічних частинах. Для створення грубого нарису вмісту сторінки, включно із верхньою та нижньою навігацією, а також елементами заголовків для розпізнання розділів вмісту, слід використовувати розділові елементи.

| Елемент                                                                                                                                                                                                                                                                                                                            | Опис                                                                                                                                                                                                                                                                                                                                                                                                      |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("address")}}                                                                                                                                                                                                                                                                                                         | Вказує на те, що HTML всередині надає контактну інформацію про особу, групу людей або організацію.                                                                                                                                                                                                                                                                                                        |
| {{HTMLElement("article")}}                                                                                                                                                                                                                                                                                                         | Представляє самодостатню композицію всередині документа, сторінки, застосунку чи сайту, котра готова для незалежного поширення або повторного використання (наприклад, шляхом передачі прав). Серед прикладів: допис на форумі, стаття в журналі чи газеті, запис у блозі, картка продукту, дописаний користувачем коментар, інтерактивний віджет або гаджет, або будь-яка інша незалежна одиниця вмісту. |
| {{HTMLElement("aside")}}                                                                                                                                                                                                                                                                                                           | Представляє частину документа, чий вміст пов'язаний з головним вмістом документа виключно непрямим чином. Остороні нерідко подаються у вигляді бічних панелей чи агітаційних блоків.                                                                                                                                                                                                                      |
| {{HTMLElement("footer")}}                                                                                                                                                                                                                                                                                                          | Представляє нижній колонтитул для свого найближчого елемента [розділового вмісту](/uk/docs/Web/HTML/Content_categories#rozdilovyi-vmist) або [розділового кореня](/uk/docs/Web/HTML/Element/Heading_Elements). Зазвичай `<footer>` містить інформацію про автора розділу, дані про авторське право чи посилання на пов'язані документи.                                                                   |
| {{HTMLElement("header")}}                                                                                                                                                                                                                                                                                                          | Представляє вступний вміст, зазвичай – групу допоміжних інструментів для знайомства чи орієнтування. Може містити якісь елементи заголовків, а також лого, форму пошуку, ім'я автора та інші елементи.                                                                                                                                                                                                    |
| [`<h1>`](/uk/docs/Web/HTML/Element/Heading_Elements), [`<h2>`](/uk/docs/Web/HTML/Element/Heading_Elements), [`<h3>`](/uk/docs/Web/HTML/Element/Heading_Elements), [`<h4>`](/uk/docs/Web/HTML/Element/Heading_Elements), [`<h5>`](/uk/docs/Web/HTML/Element/Heading_Elements), [`<h6>`](/uk/docs/Web/HTML/Element/Heading_Elements) | Представляють шість рівнів заголовків розділів. `<h1>` - найвищий рівень розділу, а `<h6>` – найнижчий.                                                                                                                                                                                                                                                                                                   |
| {{HTMLElement("hgroup")}}                                                                                                                                                                                                                                                                                                          | Представляє заголовок, згрупований з певним вторинним вмістом, як то підзаголовками, додатковою назвою чи слоганом.                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("main")}}                                                                                                                                                                                                                                                                                                            | Представляє центральний вміст тіла документа. Область головного вмісту складається зі вмісту, котрий безпосередньо пов'язаний або розкриває центральну тему документа або центральну функціональність застосунку.                                                                                                                                                                                         |
| {{HTMLElement("nav")}}                                                                                                                                                                                                                                                                                                             | Представляє розділ сторінки, чиє призначення – містити навігаційні посилання, як в межах поточного документа, так і до інших. Типові приклади розділів навігації: меню, блок вмісту, покажчик.                                                                                                                                                                                                            |
| {{HTMLElement("section")}}                                                                                                                                                                                                                                                                                                         | Представляє узагальнений окремий розділ документа, котрий не може бути представлений більш конкретним семантичним елементом. Розділи завжди повинні мати заголовки, окрім небагатьох винятків.                                                                                                                                                                                                            |
| {{HTMLElement("search")}}                                                                                                                                                                                                                                                                                                          | Представляє частину, що містить набір контрольних елементів або інший вміст, що стосується виконання пошуку або операцію фільтрування.                                                                                                                                                                                                                                                                    |

## Текстовий вміст

Для впорядкування блоків чи розділів вмісту, розташованого між початковим {{HTMLElement("body")}} та завершальним `</body>` тегами, слід використовувати елементи текстового вмісту. Що важливо для {{Glossary("accessibility", "доступності")}} та {{Glossary("SEO")}}, ці елементи дають змогу розпізнати призначення або структуру такого вмісту

| Елемент                       | Опис                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("blockquote")}} | Позначає те, що текст всередині є розлогою цитатою. Зазвичай візуально виділяється відступом. URL джерела цитування може бути вказана за допомогою атрибута `cite`, натомість текстове представлення джерела може бути задане елементом {{HTMLElement("cite")}}.                                                                                                                                                        |
| {{HTMLElement("dd")}}         | Вміщає опис, визначення або значення попереднього терміну ({{HTMLElement("dt")}}) у списку визначень.                                                                                                                                                                                                                                                                                                                   |
| {{HTMLElement("div")}}        | Узагальнений контейнер для потокового вмісту. Не впливає на вміст чи компонування, поки не оформлений якось за допомогою CSS (поки, наприклад, не застосовані певні стилі безпосереднього до нього, або поки до його батьківського елемента не застосована якась модель компонування штибу {{glossary("Flexbox", "flexbox")}}).                                                                                         |
| {{HTMLElement("dl")}}         | Представляє список визначень. Цей елемент огортає список груп термінів (заданих за допомогою елемента {{HTMLElement("dt")}}) й визначень (у вигляді елементів {{HTMLElement("dd")}}). Загальноприйняте використання цього елемента – реалізація глосарія або виведення метаданих (списку пар ключ-значення).                                                                                                            |
| {{HTMLElement("dt")}}         | Задає термін у списку описів або визначень, а отже – повинен вживатися всередині елементів {{HTMLElement("dl")}}. Зазвичай після нього стоїть елемент {{HTMLElement("dd")}}, проте кілька елементів `<dt>`, що стоять підряд, вказують на те, що всі вони визначені єдиним наступним елементом {{HTMLElement("dd")}}.                                                                                                   |
| {{HTMLElement("figcaption")}} | Представляє підпис або легенду, що описує решту вмісту свого батьківського елемента {{HTMLElement("figure")}}.                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("figure")}}     | Представляє самодостатній вміст, можливо – із необов'язковим підписом, заданим елементом {{HTMLElement("figcaption")}}. Рисунок, його підпис та вміст – вважаються єдиним цілим.                                                                                                                                                                                                                                        |
| {{HTMLElement("hr")}}         | Представляє тематичний розрив між елементами рівня абзацу – наприклад, зміну сцени в історії, або зміну теми в розділі.                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("li")}}         | Представляє елемент списку. Повинен міститися в батьківському елементі: упорядкованому списку ({{HTMLElement("ol")}}), невпорядкованому списку ({{HTMLElement("ul")}}) або меню ({{HTMLElement("menu")}}). У меню й невпорядкованих списках елементи зазвичай виводяться за допомогою маркерів списку. У впорядкованих списках вони зазвичай виводяться з нумерацією зліва, у порядку зростання, номерами або літерами. |
| {{HTMLElement("menu")}}       | Семантична альтернатива {{HTMLElement("ul")}}, котра, однак, обробляється браузерами (й виводиться в дереві доступності) так само, як {{HTMLElement("ul")}}. Представляє невпорядкований список елементів (представлених елементами {{HTMLElement("li")}}).                                                                                                                                                             |
| {{HTMLElement("ol")}}         | Представляє упорядкований список елементів, котрий здебільшого виводиться як список з нумерацією.                                                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("p")}}          | Представляє абзац. Абзаци здебільшого представляються візуальними засобами як блоки тексту, відділені від сусідніх блоків порожніми рядками чи відступом першого рядка, однак абзаци HTML можуть бути будь-яким структурним групуванням пов'язаного вмісту, як то зображень або полів форми.                                                                                                                            |
| {{HTMLElement("pre")}}        | Представляє наперед відформатований текст, котрий повинен бути представлений саме так, як він вписаний у файл HTML. Зазвичай цей текст зображається з використанням непропорційного, тобто [моноширинного](https://uk.wikipedia.org/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D1%88%D0%B8%D1%80%D0%B8%D0%BD%D0%BD%D0%B8%D0%B9_%D1%88%D1%80%D0%B8%D1%84%D1%82), шрифту. Пробіли всередині цього елемента виводяться як є.            |
| {{HTMLElement("ul")}}         | Представляє невпорядкований список елементів, що здебільшого виводиться як список з маркерами.                                                                                                                                                                                                                                                                                                                          |

## Семантичні текстові елементи

Для оголошення значення, структури чи стилю слова, рядка або будь-якого довільного шматка тексту слід використовувати семантичні текстові елементи.

| Елемент                   | Опис                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("a")}}      | Вкупі з атрибутом `href` – утворює гіперпосилання на вебсторінки, файли, адреси електронної пошти, місця на поточній сторінці або що завгодно, на що може вказувати URL.                                                                                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("abbr")}}   | Представляє абревіатуру або акронім.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| {{HTMLElement("b")}}      | Використовується для привертання уваги читача до вмісту елемента, котрому в решті не надана особлива вага. Раніше цей елемент був відомий як елемент грубого шрифту, і більшість браузерів досі виводять текст такого елемента грубим шрифтом. Проте не слід використовувати `<b>` для оформлення тексту або надання ваги. Коли потрібно виділити текст грубим шрифтом, слід використовувати властивість CSS {{cssxref("font-weight")}}. При потребі позначити елемент як елемент особливої ваги, слід використовувати елемент strong.                           |
| {{HTMLElement("bdi")}}    | Каже алгоритмові двонапрямленості браузера обробляти текст всередині окремо від тексту навколо. Це особливо корисно, коли вебсайт динамічно вставляє певний текст і не знає напряму письма тексту, що вставляється.                                                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("bdo")}}    | Відкидає поточний напрям письма в тексті, і текст всередині елемента виводиться з інакшим напрямом письма.                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("br")}}     | Породжує розрив рядка всередині тексту (повернення каретки). Корисний при записі вірша або адреси, коли поділ на рядки є важливим.                                                                                                                                                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("cite")}}   | Використовується для позначення назви цитованої праці. Посилання може бути в формі абревіатури, згідно з доречними для контексту умовностями щодо метаданих цитування.                                                                                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("code")}}   | Виводить свій вміст оформленим у стилі, котрий має на меті показати, що текст є коротким уривком комп'ютерного коду. Усталено текст вмісту виводиться за допомогою усталеного моноширинного шрифту користувацького агента.                                                                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("data")}}   | Зв'язує даний шматок вмісту з машинно-читабельним перекладом. Якщо цей вміст стосується часу або дати, то повинен використовуватися елемент time.                                                                                                                                                                                                                                                                                                                                                                                                                |
| {{HTMLElement("dfn")}}    | Вживається на позначення терміну, що визначається, в межах контексту фрази або речення. Визначенням цього терміну вважається елемент-предок {{HTMLElement("p")}}, пара {{HTMLElement("dt")}} і {{HTMLElement("dd")}} або найближчий розділовий предок елемента `<dfn>`.                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("em")}}     | Позначає текст, на котрому наголошено. Елементи `<em>` можуть бути вкладеними одне в одного, і тоді кожний рівень вкладеності позначає посилення наголошеності.                                                                                                                                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("i")}}      | Представляє масив тексту, який чомусь відділений від звичайного тексту, як то ідіоматичний текст, технічні терміни, таксономічні позначення тощо. Історично такий текст виводиться за допомогою курсивного (italicized) шрифту, що є вихідною причиною назви цього елемента – `<i>`.                                                                                                                                                                                                                                                                             |
| {{HTMLElement("kbd")}}    | Представляє уривок тексту, що позначає текст, введений з клавіатури, голосового вводу або будь-якого іншого пристрою введення тексту. Заведено, що користувацький агент усталено виводить вміст елемента `<kbd>` за допомогою свого усталеного моноширинного шрифту, попри те, що цього не вимагає стандарт HTML.                                                                                                                                                                                                                                                |
| {{HTMLElement("mark")}}   | Представляє текст, котрий позначений або виділений для потреб посилання або індексації, у зв'язку з вагою виділеного вислову в навколишньому контексті.                                                                                                                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("q")}}      | Позначає те, що текст всередині є короткою вбудованою цитатою. Більшість сучасних браузерів реалізовують його шляхом постановки тексту в лапки. Цей елемент створений для коротких цитат, що не вимагають виділення абзацом; для розлогих цитат слід використовувати елемент {{HTMLElement("blockquote")}}.                                                                                                                                                                                                                                                      |

По одному елементу `<rp>` повинно стояти навколо і початкових, і кінцевих дужок, що стоять навколо елемента {{HTMLElement("rt")}}, котрий містить текст анотації.                                                                                                                                                                                                                                            |

Елемент `<rt>` завжди повинен стояти всередині елемента {{HTMLElement("ruby")}}.                                                                                                                                                                                                                                                                                                                                |
| {{HTMLElement("ruby")}}   | Представляє невеликі анотації, що виводяться над, під або після тексту і зазвичай використовуються для виведення вимови східноазійських символів. | {{HTMLElement("s")}}      | Виводить текст закресленим, тобто з лінією посередині. Елемент `<s>` варто використовувати для представлення речей, котрі більше не актуальні або більше не точні. Проте `<s>` не є доцільним при позначенні редагувань документа, – для цього слід використовувати елементи del та ins.                                                                                                                                                                                                                                                                         |

Усталено виводить текст усередині себе на один ступінь меншим, як то `x-small` замість `small`.                                                                                                                                                                                                                                                                                                             |

| {{HTMLElement("sub")}}    | Задає вбудований текст, котрий виводиться як підрядковий суто через типографські міркування. Зазвичай надрядковий текст виводиться з базовою лінією, що вища за базову лінію звичайного тексту, меншими літерами.                                                                                                                                                                                                                                                                                                                                                |

Може містити атрибут datetime для переводу дат у машинно-читабельний формат, даючи пошуковим рушіям змогу видавати кращі результати, а також змогу реалізовувати власні можливості, як то нагадування.                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("u")}}      | Представляє уривок вбудованого тексту, котрий повинен виводитись так, щоб було видно, що він має нетекстову анотацію. Усталено виводиться з простим суцільним підкресленням, але це можна змінити за допомогою CSS.                                                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("var")}}    | Представляє назву змінної в математичному виразі або контексті програмування. Зазвичай виводиться за допомогою курсивної версії поточної гарнітури шрифту, хоч така поведінка й залежить від браузера.                                                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("wbr")}}    | Представляє нагоду розірвати слово – позицію в тексті, в якій браузер може (необов'язково) розірвати рядок, хоч його правила з розриву рядків без цього не робили б розриву в цьому місці.                                                                                                                                                                                                                                                                                                                                                                       |

## Зображення та мультимедіа

HTML підтримує різноманітні мультимедійні ресурси, наприклад, зображення, аудіо та відео.

| Елемент                  | Опис                                                                                                                                                                                                                                                                                                                                     |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("area")}}  | Задає область всередині карти зображення, котра має наперед визначені зони для клацання. _Карта зображення_ дає змогу пов'язувати геометричні зони на зображенні з {{Glossary("Hyperlink", "гіперпосиланнями")}}.                                                                                                                        |
| {{HTMLElement("audio")}} | Використовується для вбудовування в документи звукового вмісту. Може містити одне або кілька джерел аудіо, представлених атрибутом `src` або елементом source: браузер обере найбільш доречне. Також може бути приймачем потокового вмісту – за допомогою {{domxref("MediaStream")}}.                                                    |
| {{HTMLElement("img")}}   | Вбудовує в документ зображення.                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("map")}}   | Використовується вкупі з елементами {{HTMLElement("area")}} для означення карти зображення (зони посилання для клацання).                                                                                                                                                                                                                |
| {{HTMLElement("track")}} | Використовується як дочірній елемент мультимедійних елементів, аудіо або відео. Дає змогу задати текстові стежини з хронометражем (або дані, упорядковані за часом), наприклад, аби автоматично показувати субтитри. Стежини форматуються в [форматі WebVTT](/uk/docs/Web/API/WebVTT_API) (файлах `.vtt`) – текстових стежинах вебвідео. |
| {{HTMLElement("video")}} | Вставляє медіапрогравач, котрий підтримує відтворення відео всередині документа. Елемент `<video>` можна так само використовувати для аудіовмісту, але елемент audio може запропонувати більш доречний користувацький досвід.                                                                                                            |

## Вбудований вміст

На додачу до звичайного мультимедійного вмісту, HTML може включати розмаїття іншого вмісту, навіть якщо з ним не завжди легко взаємодіяти.

| Елемент                    | Опис                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("embed")}}   | Вставляє зовнішній вміст у задане місце документа. Такий вміст надається зовнішнім застосунком або іншим джерелом інтерактивного вмісту, як то плагін браузера.                                                                                                                                                                                                                                                                                        |
| {{HTMLElement("iframe")}}  | Представляє вкладений контекст перегляду, що вбудовує іншу сторінку HTML в поточну.                                                                                                                                                                                                                                                                                                                                                                    |
| {{HTMLElement("object")}}  | Представляє зовнішній ресурс, котрий може оброблятися як зображення, вкладений контекст перегляду, або ресурс, що повинен оброблятися плагіном.                                                                                                                                                                                                                                                                                                        |
| {{HTMLElement("picture")}} | Містить нуль або більше елементів {{HTMLElement("source")}}, а також один елемент {{HTMLElement("img")}}, аби запропонувати різні варіанти зображення для різних пристроїв та варіантів виведення.                                                                                                                                                                                                                                                     |
| {{HTMLElement("portal")}}  | Дає змогу вбудувати іншу сторінку HTML у поточну з міркувань плавнішого переходу до наступних сторінок.                                                                                                                                                                                                                                                                                                                                                |
| {{HTMLElement("source")}}  | Задає декілька медіаресурсів для картинки або елемента аудіо чи відео. Це пустий елемент, тобто він не має вмісту і не має завершального тега. Часто вживається для надання одного медіавмісту в кількох форматах файлів, аби запропонувати сумісність з широким діапазоном браузерів, на основі їхньої різної підтримки [форматів зображень файлів](/uk/docs/Web/Media/Formats/Image_types) і [форматів медійних файлів](/uk/docs/Web/Media/Formats). |

## SVG та MathML

Можна вбудовувати [SVG](/uk/docs/Web/SVG) та [MathML](/uk/docs/Web/MathML) вміст в HTML документи за допомогою елементів {{SVGElement("svg")}} і {{MathMLElement("math")}}.

| Елемент                   | Опис                                                                                                                                                                                                                                                                 |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{SVGElement("svg")}}     | Контейнер, що оголошує нові систему координат та [область перегляду](/uk/docs/Web/SVG/Attribute/viewBox). Він використовується як крайній зовнішній елемент документів SVG, але також може використовуватись для вбудовування фрагмента SVG у документ SVG або HTML. |
| {{MathMLElement("math")}} | Елемент вищого рівня у MathML. В нього повинен бути загорнутий кожний дійсний примірник MathML. Крім цього, не можна вставляти один елемент `<math>` в інший, але можна мати довільну кількість дочірніх елементів усередині `<math>`.                               |

## Сценарії

Для створення динамічного вмісту та вебзастосунків HTML підтримує використання мов сценаріїв, перш за все – JavaScript. Певні елементи слугують для підтримки такої функціональності.

| Елемент                     | Опис                                                                                                                                                                                                                                                                                        |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("canvas")}}   | Контейнерний елемент для використання або вкупі з [API сценаріїв полотна](/uk/docs/Web/API/Canvas_API), або [WebGL API](/uk/docs/Web/API/WebGL_API), для малювання графіки й анімацій.                                                                                                      |
| {{HTMLElement("noscript")}} | Оголошує розділ HTML, котрий буде вставлений, якщо на сторінці не підтримується певний тип сценарію, або якщо сценарії наразі вимкнені в браузері.                                                                                                                                          |
| {{HTMLElement("script")}}   | Використовується для вбудовування виконуваного коду або даних; зазвичай використовується для вбудовування або підключення коду JavaScript. Також `<script>` може вживатися вкупі з іншими мовами, як то мова програмування шейдерів GLSL [WebGL](/uk/docs/Web/API/WebGL_API), а також JSON. |

## Розмежування редагувань

Ці елементи вказують на те, що певні частини тексту були змінені

| Елемент                | Опис                                                                                                                                                                                                                                                                          |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("del")}} | Представляє уривок тексту, що був видалений з документа. Може використовуватись при візуалізації "відстеження змін" або інформації про порівняння вихідного коду, наприклад. Елемент `<ins>` може вживатись як протилежний: для позначення тексту, що був доданий в документ. |
| {{HTMLElement("ins")}} | Представляє уривок тексту, що був доданий в документ. Елемент `<del>` можна використовувати, аби в подібний спосіб представити уривок тексту, що був видалений з документа.                                                                                                   |

## Форми

HTML надає низку елементів, що можуть разом використовуватись для створення форм, котрі користувач може заповнювати та надсилати вебсайтові або застосункові. Є купа докладнішої інформації про це, доступної у [Настановах з форм HTML](/uk/docs/Learn/Forms).

| Елемент                     | Опис                                                                                                                                                                                                                                                                                                                                                                      |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("button")}}   | Інтерактивний елемент, що активується користувачем – за допомогою миші, клавіатури, пальця, голосової команди або іншої допоміжної технології. Бувши активованим, цей елемент виконує дію, наприклад, подання [форми](/uk/docs/Learn/Forms) чи відкривання діалогу.                                                                                                       |
| {{HTMLElement("datalist")}} | Містить низку елементів {{HTMLElement("option")}}, що представляють допустимі або рекомендовані варіанти, доступні для вибору в інших контрольних елементах.                                                                                                                                                                                                              |
| {{HTMLElement("fieldset")}} | Використовується для групування декількох контрольних елементів, а також їхніх підписів ({{HTMLElement("label")}}) всередині вебформи.                                                                                                                                                                                                                                    |
| {{HTMLElement("form")}}     | Представляє розділ документа, що містить інтерактивні контрольні елементи для подання інформації.                                                                                                                                                                                                                                                                         |
| {{HTMLElement("input")}}    | Використовується для створення інтерактивних контрольних елементів у вебформах, щоб приймати дані від користувача; доступний широкий асортимент типів даних і контрольних віджетів, залежний від пристрою й користувацького агента. Елемент `<input>` – один з найпотужніших комплексів у всій HTML, у зв'язку з великою кількістю комбінацій типів введення й атрибутів. |

| {{HTMLElement("meter")}}    | Представляє або скалярне значення в межах відомого діапазону, або дробове значення.                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("optgroup")}} | Утворює групу варіантів всередині елемента {{HTMLElement("select")}}.                                                                                                                                                                                                                                                                                                     |
| {{HTMLElement("option")}}   | Використовується для оголошення елемента всередині елемента select, {{HTMLElement("optgroup")}} або {{HTMLElement("datalist")}}. Сам по собі елемент `<option>` може представляти елементи меню в спливних вікнах та інших списках елементів у документі HTML.                                                                                                            |
| {{HTMLElement("output")}}   | Контейнерний елемент, в який сайт або застосунок може вивести результати обчислень або наслідок дій користувача.                                                                                                                                                                                                                                                          |
| {{HTMLElement("progress")}} | Виводить індикатор, що демонструє поступ завершення задачі, здебільшого виводиться як смуга прогресу.                                                                                                                                                                                                                                                                     |
| {{HTMLElement("select")}}   | Представляє контрольний елемент, що пропонує меню з варіантів.                                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("textarea")}} | Представляє багаторядковий контрольний елемент для редагування простого тексту, корисний, аби дати користувачам змогу ввести суттєву кількість тексту довільної форми, наприклад, коментар у формі відгуку або зворотного зв'язку.                                                                                                                                        |

## Інтерактивні елементи

HTML пропонує палітру елементів, що допоміжні у створенні інтерактивних об'єктів користувацького інтерфейсу.

| Елемент                    | Опис                                                                                                                                                                                                      |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("details")}} | Створює віджет розгортання, в якому інформація видима лише тоді, коли він переключений в стан "відкритості". Повинен бути наданий підсумок або підпис – за допомогою елемента {{HTMLElement("summary")}}. |
| {{HTMLElement("dialog")}}  | Представляє діалогове вікно або інший інтерактивний компонент, як то закривне сповіщення, інспектор або підвікно.                                                                                         |
| {{HTMLElement("summary")}} | Задає підсумок, підпис або легенду для рамки розкриття елемента details. Клацання елемента `<summary>` перемикає стан батьківського елемента {{HTMLElement("details")}} між відкритістю й закритістю.     |

## Вебкомпоненти

Вебкомпоненти – пов'язана із HTML технологія, що дає змогу, по суті, створювати та використовувати користувацькі елементи так, ніби вони є звичайним HTML. На додачу – можна створювати користувацькі версії стандартних елементів HTML.

| Елемент                     | Опис                                                                                                                                                                                                                                   |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("slot")}}     | Частина технологічного блоку [Вебкомпонентів](/uk/docs/Web/API/Web_components); цей елемент є слотом всередині вебкомпонента, котрий можна заповнити власною розміткою, що дає змогу створювати окремі дерева DOM і виводити їх разом. |
| {{HTMLElement("template")}} | Механізм для зберігання HTML, що не повинен відображатися одразу після завантаження сторінки, проте може створити свій примірник згодом під час виконання JavaScript.                                                                  |

## Невживані та нерекомендовані елементи

> **Примітка:** Це старі елементи HTML, що не повинні використовуватися. **Їх ніколи не слід використовувати у нових проєктах, а також слід заміняти у старих проєктах настільки скоро, наскільки можливо.** Вони перелічені виключно заради повноти інформації.

| ---------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("acronym")}}   | Дає розробникам змогу чітко позначити послідовність символів, що утворює акронім або абревіатуру.                                                                                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("big")}}       | Візуалізує текст всередині з розміром шрифту на ступінь більше за текст навколо (`medium`, наприклад, стає `large`). Цей розмір обмежений найбільшим дозволеним браузером розмір шрифту.                                                                                                                                                                                                                                                                        |
| {{HTMLElement("center")}}    | Виводить свій вміст, рядковий або блокового рівня, відцентрованим горизонтально всередині свого контейнерного елемента.                                                                                                                                                                                                                                                                                                                                         |
| `<content>`                  | Застаріла частина блоку технологій [Вебкомпонентів](/uk/docs/Web/API/Web_components), що використовувалася всередині [тіньової DOM](/uk/docs/Web/API/Web_components/Using_shadow_DOM) як точка вставки, і для використання у звичайному HTML не планувалася. Тепер цей елемент замінений елементом {{HTMLElement("slot")}}, котрий створює точку в DOM, в якій може бути вставлено тіньову DOM. Натомість слід розглянути використання {{HTMLElement("slot")}}. |
| {{HTMLElement("dir")}}       | Контейнер для директорії файлів чи тек, можливо – зі стилями та піктограмами, застосованими користувацьким агентом. Не слід використовувати цей застарілий елемент, замість нього слід вживати елемент {{HTMLElement("ul")}} для списків, включно зі списками файлів.                                                                                                                                                                                           |

Елемент frame повинен використовуватися всередині {{HTMLElement("frameset")}}                                                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("frameset")}}  | Використовується, аби вміщати елементи {{HTMLElement("frame")}}.                                                                                                                                                                                                                                                                                                                                                                                                |
| {{HTMLElement("image")}}     | Стародавній та погано підтримуваний попередник елемента {{HTMLElement("img")}}. Не повинен використовуватися.                                                                                                                                                                                                                                                                                                                                                   |

Можна контролювати те, що відбувається, коли текст досягає країв своєї області вмісту, за допомогою атрибутів цього елемента.                                                                                                                                                                                                                                                                           |
