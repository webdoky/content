- –ü–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è: `\b`, `\-`, [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Character_class_escape), [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤ Unicode](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Unicode_character_class_escape) —Ç–∞ —ñ–Ω—à—ñ [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è —Å–∏–º–≤–æ–ª—ñ–≤](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Character_escape).

–¶—ñ –∑–∞–ø–∏—Å–∏ –º–æ–∂—É—Ç—å –∑—É—Å—Ç—Ä—ñ—á–∞—Ç–∏—Å—è –±—É–¥—å-—è–∫—É –∫—ñ–ª—å–∫—ñ—Å—Ç—å —Ä–∞–∑—ñ–≤, —ñ –Ω–∞–±–æ—Ä–∏ —Å–∏–º–≤–æ–ª—ñ–≤, –∫–æ—Ç—Ä—ñ –≤–æ–Ω–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç—å, –æ–±'—î–¥–Ω—É—é—Ç—å—Å—è. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `/[a-zA-Z0-9]/` –¥–∞—î –∑–±—ñ–≥ –∑ –±—É–¥—å-—è–∫–æ—é –ª—ñ—Ç–µ—Ä–æ—é –∞–±–æ —Ü–∏—Ñ—Ä–æ—é.

–ù–∞ –≤—ñ–¥–º—ñ–Ω—É –≤—ñ–¥ —ñ–Ω—à–∏—Ö —á–∞—Å—Ç–∏–Ω —Ä–µ–≥—É–ª—è—Ä–Ω–æ–≥–æ –≤–∏—Ä–∞–∑—É, –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ —Ç–ª—É–º–∞—á–∞—Ç—å –±—ñ–ª—å—à—ñ—Å—Ç—å —Å–∏–º–≤–æ–ª—ñ–≤ [–±—É–∫–≤–∞–ª—å–Ω–æ](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Literal_character) —Ç–∞ –º–∞—é—Ç—å –º–µ–Ω—à–µ –æ–±–º–µ–∂–µ–Ω—å —â–æ–¥–æ —Å–∏–º–≤–æ–ª—ñ–≤, –∫–æ—Ç—Ä—ñ –º–æ–∂—É—Ç—å –≤–º—ñ—â–∞—Ç–∏. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `.` ‚Äì —Ü–µ –±—É–∫–≤–∞–ª—å–Ω–∏–π —Å–∏–º–≤–æ–ª –∫—Ä–∞–ø–∫–∏, –∞ –Ω–µ [–¥–∂–æ–∫–µ—Ä](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Wildcard). –Ñ–¥–∏–Ω—ñ —Å–∏–º–≤–æ–ª–∏, —â–æ –Ω–µ –º–æ–∂—É—Ç—å –∑—É—Å—Ç—Ä—ñ—á–∞—Ç–∏—Å—è –±—É–∫–≤–∞–ª—å–Ω–æ, —Ü–µ `\`, `]` —ñ `-`.

- –£ –∫–ª–∞—Å–∞—Ö —Å–∏–º–≤–æ–ª—ñ–≤ –ø—ñ–¥—Ç—Ä–∏–º—É—î—Ç—å—Å—è –±—ñ–ª—å—à—ñ—Å—Ç—å –ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç–µ–π –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è, –æ–∫—Ä—ñ–º `\b`, `\B` —ñ [–∑–≤–æ—Ä–æ—Ç–Ω–∏—Ö –ø–æ—Å–∏–ª–∞–Ω—å](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Backreference). `\b` –ø–æ–∑–Ω–∞—á–∞—î —Å–∏–º–≤–æ–ª —Ä–µ–≤–µ—Ä—Å—É, –∞ –Ω–µ [–∫—Ä–∞–π —Å–ª–æ–≤–∞](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Word_boundary_assertion), —Ç–∏–º —á–∞—Å–æ–º —ñ–Ω—à—ñ –¥–≤–∞ –∫–ª–∞—Å–∏ –≤–∏–∫–ª–∏–∫–∞—é—Ç—å —Å–∏–Ω—Ç–∞–∫—Å–∏—á–Ω—ñ –ø–æ–º–∏–ª–∫–∏. –©–æ–± –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ `\` –±—É–∫–≤–∞–ª—å–Ω–æ, —Ü–µ–π —Å–∏–º–≤–æ–ª —Å–ª—ñ–¥ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏: `\\`.
- –°–∏–º–≤–æ–ª `]` –ø–æ–∑–Ω–∞—á–∞—î –∫—ñ–Ω–µ—Ü—å –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤. –©–æ–± –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ –π–æ–≥–æ –±—É–∫–≤–∞–ª—å–Ω–æ, –π–æ–≥–æ —Å–ª—ñ–¥ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏: `\]`.
- –°–∏–º–≤–æ–ª –¥–µ—Ñ—ñ—Å–∞ (`-`), –∫–æ–ª–∏ –≤–∂–∏–≤–∞—î—Ç—å—Å—è –º—ñ–∂ –¥–≤–æ–º–∞ —Å–∏–º–≤–æ–ª–∞–º–∏, –ø–æ–∑–Ω–∞—á–∞—î –¥—ñ–∞–ø–∞–∑–æ–Ω. –ö–æ–ª–∏ –≤—ñ–Ω –∑—É—Å—Ç—Ä—ñ—á–∞—î—Ç—å—Å—è –Ω–∞ –ø–æ—á–∞—Ç–∫—É –∞–±–æ –≤ –∫—ñ–Ω—Ü—ñ –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤, –≤—ñ–Ω —î –±—É–∫–≤–∞–ª—å–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º. –í—ñ–Ω —Ç–∞–∫–æ–∂ —î –±—É–∫–≤–∞–ª—å–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º, –∫–æ–ª–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –≤ –º–µ–∂–∞—Ö –¥—ñ–∞–ø–∞–∑–æ–Ω—É. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `[a-]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ `a` —ñ `-`, `[!--]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ –≤—ñ–¥ `!` –¥–æ `-`, –∞ `[--9]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ –≤—ñ–¥ `-` –¥–æ `9`. –¢–∞–∫–æ–∂ –º–æ–∂–Ω–∞ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏ –¥–µ—Ñ—ñ—Å —è–∫ `\-`, —â–æ–± –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –π–æ–≥–æ –±—É–¥—å-–¥–µ –±—É–∫–≤–∞–ª—å–Ω–æ.

[–õ–µ–∫—Å–∏—á–Ω–∞ –≥—Ä–∞–º–∞—Ç–∏–∫–∞](/uk/docs/Web/JavaScript/Reference/Lexical_grammar#literaly-rehuliarnykh-vyraziv) –≤–∏–∫–æ–Ω—É—î –¥—É–∂–µ –≥—Ä—É–±–∏–π —Ä–æ–∑–±—ñ—Ä –ª—ñ—Ç–µ—Ä–∞–ª—ñ–≤ —Ä–µ–≥—É–ª—è—Ä–Ω–∏—Ö –≤–∏—Ä–∞–∑—ñ–≤, —Ç–æ–º—É —â–æ –≤–æ–Ω–∞ –Ω–µ –∑–∞–≤–µ—Ä—à—É—î –ª—ñ—Ç–µ—Ä–∞–ª —Ä–µ–≥—É–ª—è—Ä–Ω–æ–≥–æ –≤–∏—Ä–∞–∑—É –Ω–∞ —Å–∏–º–≤–æ–ª—ñ `/`, –∫–æ—Ç—Ä–∏–π –∑—É—Å—Ç—Ä—ñ—á–∞—î—Ç—å—Å—è –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤. –¶–µ –æ–∑–Ω–∞—á–∞—î, —â–æ `/[/]/` —î –¥–æ–ø—É—Å—Ç–∏–º–∏–º –±–µ–∑ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ—Å—Ç—ñ –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è `/`.

–ö—Ä–∞—ó –¥—ñ–∞–ø–∞–∑–æ–Ω—É —Å–∏–º–≤–æ–ª—ñ–≤ –Ω–µ –ø–æ–≤–∏–Ω–Ω—ñ –∑–∞–¥–∞–≤–∞—Ç–∏ –±—ñ–ª—å—à –Ω—ñ–∂ –æ–¥–∏–Ω —Å–∏–º–≤–æ–ª, —â–æ –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è, —è–∫—â–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Character_class_escape). –ù–∞–ø—Ä–∏–∫–ª–∞–¥:

```js

```

–í —Ä–µ–∂–∏–º—ñ –±–µ–∑ [unicode](/uk/docs/Web/JavaScript/Reference/Global_Objects/RegExp/unicode) –¥—ñ–∞–ø–∞–∑–æ–Ω–∏ —Å–∏–º–≤–æ–ª—ñ–≤, —É —è–∫–∏—Ö –æ–¥–Ω–∞ –∑ –º–µ–∂ —î –∫–ª–∞—Å–æ–º —Å–∏–º–≤–æ–ª—ñ–≤, –∑–º—É—à—É—é—Ç—å `-` —Å—Ç–∞—Ç–∏ –±—É–∫–≤–∞–ª—å–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º. –¶–µ [–Ω–µ—Ä–µ–∫–æ–º–µ–Ω–¥–æ–≤–∞–Ω–∏–π –∑–∞–ø–∏—Å, –Ω–µ–æ–±—Ö—ñ–¥–Ω–∏–π –¥–ª—è –≤–µ–±—Å—É–º—ñ—Å–Ω–æ—Å—Ç—ñ](/uk/docs/Web/JavaScript/Reference/Deprecated_and_obsolete_features#regexp), —ñ –Ω–∞ –Ω—å–æ–≥–æ –Ω–µ —Å–ª—ñ–¥ –ø–æ–∫–ª–∞–¥–∞—Ç–∏—Å—è.


–í —Ä–µ–∂–∏–º—ñ –±–µ–∑ [unicode](/uk/docs/Web/JavaScript/Reference/Global_Objects/RegExp/unicode) —Ä–µ–≥—É–ª—è—Ä–Ω—ñ –≤–∏—Ä–∞–∑–∏ —Ç–ª—É–º–∞—á–∞—Ç—å—Å—è —è–∫ –ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ —Å–∏–º–≤–æ–ª—ñ–≤ –ë–∞–∑–æ–≤–æ–≥–æ –±–∞–≥–∞—Ç–æ–º–æ–≤–Ω–æ–≥–æ –ø–ª–∞–Ω—É. –¢–∞–∫–∏–º —á–∏–Ω–æ–º, —Å—É—Ä–æ–≥–∞—Ç–Ω—ñ –ø–∞—Ä–∏ –≤ –∫–ª–∞—Å–∞—Ö —Å–∏–º–≤–æ–ª—ñ–≤ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç—å –¥–≤–∞ —Å–∏–º–≤–æ–ª–∏ –∑–∞–º—ñ—Å—Ç—å –æ–¥–Ω–æ–≥–æ.

```js
/[üòÑ]/.test("\ud83d"); // true
```

–ù–∞–≤—ñ—Ç—å —è–∫—â–æ –ø–∞—Ç–µ—Ä–Ω [—ñ–≥–Ω–æ—Ä—É—î —Ä–µ–≥—ñ—Å—Ç—Ä](/uk/docs/Web/JavaScript/Reference/Global_Objects/RegExp/ignoreCase), —Ç–æ —Ä–µ–≥—ñ—Å—Ç—Ä –¥–≤–æ—Ö –∫—ñ–Ω—Ü—ñ–≤ –¥—ñ–∞–ø–∞–∑–æ–Ω—É —î –≤–∞–∂–ª–∏–≤–∏–º –ø—Ä–∏ –∑'—è—Å—É–≤–∞–Ω–Ω—ñ —Ç–æ–≥–æ, —è–∫—ñ —Å–∏–º–≤–æ–ª–∏ –Ω–∞–ª–µ–∂–∞—Ç—å –¥–æ —Ü—å–æ–≥–æ –¥—ñ–∞–ø–∞–∑–æ–Ω—É. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, –ø–∞—Ç–µ—Ä–Ω `/[E-F]/i` –¥–∞—î –∑–±—ñ–≥ –ª–∏—à–µ –∑ `E`, `F`, `e` —Ç–∞ `f`, –∞ –ø–∞—Ç–µ—Ä–Ω `/[E-f]/i` –¥–∞—î –∑–±—ñ–≥ –∑ —É—Å—ñ–º–∞ –≤–µ–ª–∏–∫–∏–º–∏ —Ç–∞ –º–∞–ª–∏–º–∏ –ª—ñ—Ç–µ—Ä–∞–º–∏ ASCII (—Ç–æ–º—É, —â–æ –ø—Ä–æ—Å—Ç—è–≥–∞—î—Ç—å—Å—è –≤—ñ–¥ `E` –¥–æ `Z` —Ç–∞ –≤—ñ–¥ `a` –¥–æ `f`), –∞ —Ç–∞–∫–æ–∂ `[`, `\`, `]`, `^`, `_` —Ç–∞ `` ` ``.

–ü—Ä–µ—Ñ—ñ–∫—Å `^` –≤ –∫–ª–∞—Å—ñ —Å–∏–º–≤–æ–ª—ñ–≤ ‚Äì –æ–±–µ—Ä—Ç–∞—î –∑–±—ñ–≥. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `[^abc]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ –±—É–¥—å-—è–∫–∏–º —Å–∏–º–≤–æ–ª–æ–º, –∫—Ä—ñ–º `a`, `b` —ñ `c`. –°–∏–º–≤–æ–ª `^` —î –±—É–∫–≤–∞–ª—å–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º, –∫–æ–ª–∏ –∑'—è–≤–ª—è—î—Ç—å—Å—è –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤ ‚Äì –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `[a^b]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ `a`, `^` —Ç–∞ `b`.

## –ü—Ä–∏–∫–ª–∞–¥–∏

### –ü–æ—à—É–∫ —à—ñ—Å—Ç–Ω–∞–¥—Ü—è—Ç–∫–æ–≤–∏—Ö —Ü–∏—Ñ—Ä

–ù–∞—Å—Ç—É–ø–Ω–∞ —Ñ—É–Ω–∫—Ü—ñ—è –∑'—è—Å–æ–≤—É—î, —á–∏ –º—ñ—Å—Ç–∏—Ç—å —Ä—è–¥–æ–∫ –¥—ñ–π—Å–Ω–µ —à—ñ—Å—Ç–Ω–∞–¥—Ü—è—Ç–∫–æ–≤–µ —á–∏—Å–ª–æ:

```js
function isHexadecimal(str) {
  return /^[0-9A-F]+$/i.test(str);
}

isHexadecimal("2F3"); // true
isHexadecimal("beef"); // true
isHexadecimal("undefined"); // false
```
