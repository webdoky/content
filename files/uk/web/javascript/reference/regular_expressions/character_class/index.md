- –ü–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è: `\b`, `\-`, [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Character_class_escape), [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤ Unicode](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Unicode_character_class_escape) —Ç–∞ —ñ–Ω—à—ñ [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è —Å–∏–º–≤–æ–ª—ñ–≤](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Character_escape).

–¶—ñ –∑–∞–ø–∏—Å–∏ –º–æ–∂—É—Ç—å –∑—É—Å—Ç—Ä—ñ—á–∞—Ç–∏—Å—è –±—É–¥—å-—è–∫—É –∫—ñ–ª—å–∫—ñ—Å—Ç—å —Ä–∞–∑—ñ–≤, —ñ –Ω–∞–±–æ—Ä–∏ —Å–∏–º–≤–æ–ª—ñ–≤, –∫–æ—Ç—Ä—ñ –≤–æ–Ω–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç—å, –æ–±'—î–¥–Ω—É—é—Ç—å—Å—è. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `/[a-zA-Z0-9]/` –¥–∞—î –∑–±—ñ–≥ –∑ –±—É–¥—å-—è–∫–æ—é –ª—ñ—Ç–µ—Ä–æ—é –∞–±–æ —Ü–∏—Ñ—Ä–æ—é.

–ü—Ä–µ—Ñ—ñ–∫—Å `^` —É –∫–ª–∞—Å—ñ —Å–∏–º–≤–æ–ª—ñ–≤ —Å—Ç–≤–æ—Ä—é—î _–¥–æ–ø–æ–≤–Ω—è–ª—å–Ω–∏–π –∫–ª–∞—Å_. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `[^abc]` –¥–∞—î –∑–±—ñ–≥ –∑ –±—É–¥—å-—è–∫–∏–º —Å–∏–º–≤–æ–ª–æ–º, –æ–∫—Ä—ñ–º `a`, `b` —ñ `c`. –°–∏–º–≤–æ–ª `^` —î –±—É–∫–≤–∞–ª—å–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º, –∫–æ–ª–∏ –∑—É—Å—Ç—Ä—ñ—á–∞—î—Ç—å—Å—è –≤ —Å–µ—Ä–µ–¥–∏–Ω—ñ –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤ ‚Äì –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `[a^b]` –¥–∞—î –∑–±—ñ–≥ –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ `a`, `^` —ñ `b`.

### –ö–ª–∞—Å —Å–∏–º–≤–æ–ª—ñ–≤ —É —Ä–µ–∂–∏–º—ñ –±–µ–∑ v

–ö–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ —É —Ä–µ–∂–∏–º—ñ –±–µ–∑ `v` —Ç–ª—É–º–∞—á–∞—Ç—å –±—ñ–ª—å—à—ñ—Å—Ç—å —Å–∏–º–≤–æ–ª—ñ–≤ [–±—É–∫–≤–∞–ª—å–Ω–æ](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Literal_character) —Ç–∞ –º–∞—é—Ç—å –º–µ–Ω—à–µ –æ–±–º–µ–∂–µ–Ω—å —â–æ–¥–æ —Å–∏–º–≤–æ–ª—ñ–≤, –∫–æ—Ç—Ä—ñ –º–æ–∂—É—Ç—å –≤–º—ñ—â–∞—Ç–∏. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `.` - —Ü–µ –±—É–∫–≤–∞–ª—å–Ω–∏–π —Å–∏–º–≤–æ–ª –∫—Ä–∞–ø–∫–∏, –∞ –Ω–µ [–¥–∂–æ–∫–µ—Ä](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Wildcard). –Ñ–¥–∏–Ω—ñ —Å–∏–º–≤–æ–ª–∏, —â–æ –Ω–µ –º–æ–∂—É—Ç—å –∑—É—Å—Ç—Ä—ñ—á–∞—Ç–∏—Å—è –±—É–∫–≤–∞–ª—å–Ω–æ, ‚Äì `\`, `]` —Ç–∞ `-`.

- –£ –∫–ª–∞—Å–∞—Ö —Å–∏–º–≤–æ–ª—ñ–≤ –ø—ñ–¥—Ç—Ä–∏–º—É—î—Ç—å—Å—è –±—ñ–ª—å—à—ñ—Å—Ç—å –µ–∫—Ä–∞–Ω—É–≤–∞–ª—å–Ω–∏—Ö –ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç–µ–π, –æ–∫—Ä—ñ–º `\b`, `\B` —Ç–∞ [–∑–≤–æ—Ä–æ—Ç–Ω–∏—Ö –ø–æ—Å–∏–ª–∞–Ω—å](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Backreference). –ü–æ—Å–ª—ñ–¥–æ–≤–Ω—ñ—Å—Ç—å `\b` –ø–æ–∑–Ω–∞—á–∞—î —Å–∏–º–≤–æ–ª —Ä–µ–≤–µ—Ä—Å—É, –∞ –Ω–µ [–º–µ–∂—É —Å–ª–æ–≤–∞](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Word_boundary_assertion), –∞ –¥–≤–∞ —ñ–Ω—à–∏—Ö –ø—Ä–∏–∑–≤–æ–¥—è—Ç—å –¥–æ —Å–∏–Ω—Ç–∞–∫—Å–∏—á–Ω–∏—Ö –ø–æ–º–∏–ª–æ–∫. –©–æ–± –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ `\` –±—É–∫–≤–∞–ª—å–Ω–æ, —Ü–µ–π —Å–∏–º–≤–æ–ª —Å–ª—ñ–¥ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏ —É –≤–∏–≥–ª—è–¥—ñ `\\`.
- –°–∏–º–≤–æ–ª `]` –≤–∫–∞–∑—É—î –Ω–∞ –∫—ñ–Ω–µ—Ü—å –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤. –©–æ–± –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –π–æ–≥–æ –±—É–∫–≤–∞–ª—å–Ω–æ, –π–æ–≥–æ —Å–ª—ñ–¥ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏ —É –≤–∏–≥–ª—è–¥—ñ `\]`.
- –°–∏–º–≤–æ–ª –¥–µ—Ñ—ñ—Å–∞ (`-`), –∫–æ–ª–∏ –≤–∂–∏—Ç–∏–π –º—ñ–∂ –¥–≤–æ–º–∞ —Å–∏–º–≤–æ–ª–∞–º–∏, –ø–æ–∑–Ω–∞—á–∞—î –¥—ñ–∞–ø–∞–∑–æ–Ω. –ö–æ–ª–∏ –≤—ñ–Ω –∑—É—Å—Ç—Ä—ñ—á–∞—î—Ç—å—Å—è –Ω–∞ –ø–æ—á–∞—Ç–∫—É –∞–±–æ –≤ –∫—ñ–Ω—Ü—ñ –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤, —Ç–æ —î –±—É–∫–≤–∞–ª—å–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º. –í—ñ–Ω —Ç–∞–∫–æ–∂ —î –±—É–∫–≤–∞–ª—å–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º, –∫–æ–ª–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è —è–∫ –º–µ–∂–∞ –¥—ñ–∞–ø–∞–∑–æ–Ω—É. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `[a-]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ `a` —Ç–∞ `-`, `[!--]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ –≤—ñ–¥ `!` –¥–æ `-`, –∞ `[--9]` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ –≤—ñ–¥ `-` –¥–æ `9`. –¢–∞–∫–æ–∂ –º–æ–∂–Ω–∞ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏ –π–æ–≥–æ —è–∫ `\-`, —è–∫—â–æ —î –ø–æ—Ç—Ä–µ–±–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –π–æ–≥–æ –±—É–∫–≤–∞–ª—å–Ω–æ —É –±—É–¥—å-—è–∫–æ–º—É –º—ñ—Å—Ü—ñ.

### –ö–ª–∞—Å —Å–∏–º–≤–æ–ª—ñ–≤ —É —Ä–µ–∂–∏–º—ñ –∑ v

–ë–∞–∑–æ–≤–∞ —ñ–¥–µ—è –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤ —É —Ä–µ–∂–∏–º—ñ `v` –∑–∞–ª–∏—à–∞—î—Ç—å—Å—è —Ç—ñ—î—é –∂: —Ç–∞–∫ —Å–∞–º–æ –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –±—ñ–ª—å—à—ñ—Å—Ç—å —Å–∏–º–≤–æ–ª—ñ–≤ –±—É–∫–≤–∞–ª—å–Ω–æ, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ `-` –¥–ª—è –ø–æ–∑–Ω–∞—á–µ–Ω–Ω—è –¥—ñ–∞–ø–∞–∑–æ–Ω—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤ —ñ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è. –û–¥–Ω—ñ—î—é –∑ –Ω–∞–π–≤–∞–∂–ª–∏–≤—ñ—à–∏—Ö –æ—Å–æ–±–ª–∏–≤–æ—Å—Ç–µ–π –ø–æ–∑–Ω–∞—á–∫–∏ `v` —î _–∑–∞–ø–∏—Å –º–Ω–æ–∂–∏–Ω_ —É –º–µ–∂–∞—Ö –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤. –Ø–∫ –≤–∂–µ –∑–≥–∞–¥—É–≤–∞–ª–æ—Å—è —Ä–∞–Ω—ñ—à–µ, –∑–≤–∏—á–∞–π–Ω—ñ –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ –º–æ–∂—É—Ç—å –≤–∏—Ä–∞–∂–∞—Ç–∏ –æ–±'—î–¥–Ω–∞–Ω–Ω—è, –∑—á—ñ–ø–ª—é—é—á–∏ –¥–æ–∫—É–ø–∏ –¥–≤–∞ –¥—ñ–∞–ø–∞–∑–æ–Ω–∏, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ `[A-Z0-9]` –¥–ª—è –ø–æ–∑–Ω–∞—á–µ–Ω–Ω—è "–æ–±'—î–¥–Ω–∞–Ω–Ω—è –º–Ω–æ–∂–∏–Ω–∏ `[A-Z]` —Ç–∞ –º–Ω–æ–∂–∏–Ω–∏ `[0-9]`". –û–¥–Ω–∞–∫ –Ω–µ–º–∞—î –ø—Ä–æ—Å—Ç–æ–≥–æ —Å–ø–æ—Å–æ–±—É –≤–∏—Ä–∞–∑–∏—Ç–∏ —ñ–Ω—à—ñ –æ–ø–µ—Ä–∞—Ü—ñ—ó –∑ –º–Ω–æ–∂–∏–Ω–∞–º–∏ —Å–∏–º–≤–æ–ª—ñ–≤, —Ç–∞–∫—ñ —è–∫ –ø–µ—Ä–µ—Ç–∏–Ω —Ç–∞ —Ä—ñ–∑–Ω–∏—Ü—è.

–ó–∞ –ø—Ä–∏—Å—É—Ç–Ω–æ—Å—Ç—ñ –ø–æ–∑–Ω–∞—á–∫–∏ `v` –ø–µ—Ä–µ—Ç–∏–Ω –≤–∏—Ä–∞–∂–∞—î—Ç—å—Å—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é `&&`, –∞ —Ä—ñ–∑–Ω–∏—Ü—è ‚Äî –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é `--`. –í—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—å –æ–±–æ—Ö —Ü–∏—Ö –∫–æ–º–±—ñ–Ω–∞—Ü—ñ–π –ø–æ–∑–Ω–∞—á–∞—î –æ–±'—î–¥–Ω–∞–Ω–Ω—è. –î–≤–∞ –æ–ø–µ—Ä–∞–Ω–¥–∏ `&&` –∞–±–æ `--` –º–æ–∂—É—Ç—å –±—É—Ç–∏ —Å–∏–º–≤–æ–ª–∞–º–∏, –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è–º–∏ —Å–∏–º–≤–æ–ª—ñ–≤, –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è–º –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤ –∞–±–æ –Ω–∞–≤—ñ—Ç—å —ñ–Ω—à–∏–º –∫–ª–∞—Å–æ–º —Å–∏–º–≤–æ–ª—ñ–≤. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, —â–æ–± –≤–∏—Ä–∞–∑–∏—Ç–∏ "—Å–∏–º–≤–æ–ª –ª—ñ—Ç–µ—Ä–∏ –∞–±–æ —Ü–∏—Ñ—Ä–∏, –∞–ª–µ –Ω–µ –ø—ñ–¥–∫—Ä–µ—Å–ª–µ–Ω–Ω—è", –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ `[\w--_]`. –ù–µ –º–æ–∂–Ω–∞ –∑–º—ñ—à—É–≤–∞—Ç–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –Ω–∞ –æ–¥–Ω–æ–º—É —Ä—ñ–≤–Ω—ñ. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `[\w&&[A-z]--_]` —î —Å–∏–Ω—Ç–∞–∫—Å–∏—á–Ω–æ—é –ø–æ–º–∏–ª–∫–æ—é. –û–¥–Ω–∞–∫, –æ—Å–∫—ñ–ª—å–∫–∏ –º–æ–∂–Ω–∞ –≤–∫–ª–∞–¥–∞—Ç–∏ –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ –æ–¥–∏–Ω –≤ –æ–¥–Ω–æ–≥–æ, –º–æ–∂–Ω–∞ —è–≤–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç–∏ `[\w&&[[A-z]--_]]` –∞–±–æ `[[\w&&[A-z]]--_]` (–æ–±–∏–¥–≤–∞ —Ü—ñ –∑–∞–ø–∏—Å–∏ –æ–∑–Ω–∞—á–∞—é—Ç—å `[A-Za-z]`). –ê–Ω–∞–ª–æ–≥—ñ—á–Ω–æ, `[AB--C]` —î –Ω–µ–¥—ñ–π—Å–Ω–∏–º, —ñ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç–∏ `[A[B--C]]` (—â–æ –æ–∑–Ω–∞—á–∞—î –ø—Ä–æ—Å—Ç–æ `[AB]`).

–£ —Ä–µ–∂–∏–º—ñ `v` [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –∫–ª–∞—Å—É —Å–∏–º–≤–æ–ª—ñ–≤ Unicode](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Unicode_character_class_escape) `\p` –º–æ–∂–µ –¥–∞–≤–∞—Ç–∏ –∑–±—ñ–≥ –∑ —Ä—è–¥–∫–∞–º–∏ —Å–∫—ñ–Ω—á–µ–Ω–Ω–æ—ó –¥–æ–≤–∂–∏–Ω–∏, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –µ–º–æ–¥–∂—ñ. –ó–∞–¥–ª—è —Å–∏–º–µ—Ç—Ä—ñ—ó, –∑–≤–∏—á–∞–π–Ω—ñ –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ —Ç–∞–∫–æ–∂ –º–æ–∂—É—Ç—å –¥–∞–≤–∞—Ç–∏ –∑–±—ñ–≥ –±—ñ–ª—å—à –Ω—ñ–∂ –∑ –æ–¥–Ω–∏–º —Å–∏–º–≤–æ–ª–æ–º. –©–æ–± –∑–∞–ø–∏—Å–∞—Ç–∏ "—Ä—è–¥–∫–æ–≤–∏–π –ª—ñ—Ç–µ—Ä–∞–ª" —É –∫–ª–∞—Å—ñ —Å–∏–º–≤–æ–ª—ñ–≤, —Ç–∞–∫–∏–π —Ä—è–¥–æ–∫ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –∑–∞–≥–æ—Ä–Ω—É—Ç–∏ –≤ `\q{...}`. –Ñ–¥–∏–Ω–∏–π –∑–∞–ø–∏—Å —Ä–µ–≥—É–ª—è—Ä–Ω–∏—Ö –≤–∏—Ä–∞–∑—ñ–≤, —è–∫–∏–π –ø—ñ–¥—Ç—Ä–∏–º—É—î—Ç—å—Å—è —Ç—É—Ç, ‚Äî —Ü–µ [–¥–∏–∑'—é–Ω–∫—Ü—ñ—è](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Disjunction) ‚Äî –∫—Ä—ñ–º —Ü—å–æ–≥–æ, `\q` –ø–æ–≤–∏–Ω–µ–Ω –ø–æ–≤–Ω—ñ—Å—Ç—é –æ—Ç–æ—á—É–≤–∞—Ç–∏ –ª—ñ—Ç–µ—Ä–∞–ª–∏ (–≤–∫–ª—é—á–Ω–æ –∑ –µ–∫—Ä–∞–Ω–æ–≤–∞–Ω–∏–º–∏ —Å–∏–º–≤–æ–ª–∞–º–∏). –¶–µ –∑–∞–±–µ–∑–ø–µ—á—É—î —Ç–µ, —â–æ –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ –º–æ–∂—É—Ç—å –¥–∞–≤–∞—Ç–∏ –∑–±—ñ–≥ –ª–∏—à–µ –∑ —Ä—è–¥–∫–∞–º–∏ —Å–∫—ñ–Ω—á–µ–Ω–Ω–æ—ó –¥–æ–≤–∂–∏–Ω–∏ –∑—ñ —Å–∫—ñ–Ω—á–µ–Ω–Ω–æ—é –∫—ñ–ª—å–∫—ñ—Å—Ç—é –º–æ–∂–ª–∏–≤–∏—Ö –≤–∞—Ä—ñ–∞–Ω—Ç—ñ–≤.

–£ –∑–≤'—è–∑–∫—É –∑ —Ç–∏–º, —â–æ –∑–∞–ø–∏—Å –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤ —Å—Ç–∞–≤ —Å–∫–ª–∞–¥–Ω—ñ—à–∏–º, –±—ñ–ª—å—à–µ —Å–∏–º–≤–æ–ª—ñ–≤ —Å—Ç–∞–ª–∏ –∑–∞—Ä–µ–∑–µ—Ä–≤–æ–≤–∞–Ω–∏–º–∏ —Ç–∞ –∑–∞–±–æ—Ä–æ–Ω–µ–Ω–∏–º–∏ –¥–ª—è –±—É–∫–≤–∞–ª—å–Ω–æ—ó –ø–æ—è–≤–∏.

- –ù–∞ –¥–æ–¥–∞—á—É –¥–æ `]` —ñ `\`, –Ω–∞—Å—Ç—É–ø–Ω—ñ —Å–∏–º–≤–æ–ª–∏ –ø–æ–≤–∏–Ω–Ω—ñ –±—É—Ç–∏ –µ–∫—Ä–∞–Ω–æ–≤–∞–Ω—ñ –≤ –∫–ª–∞—Å–∞—Ö —Å–∏–º–≤–æ–ª—ñ–≤, —è–∫—â–æ –≤–æ–Ω–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç—å –±—É–∫–≤–∞–ª—å–Ω—ñ —Å–∏–º–≤–æ–ª–∏: `(`, `)`, `[`, `{`, `}`, `/`, `-`, `|`. –¶–µ–π —Å–ø–∏—Å–æ–∫ –¥–µ—â–æ –ø–æ–¥—ñ–±–Ω–∏–π –¥–æ —Å–ø–∏—Å–∫—É [—Å–∏–Ω—Ç–∞–∫—Å–∏—á–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Literal_character), –∑–∞ –≤–∏–Ω—è—Ç–∫–æ–º —Ç–æ–≥–æ, —â–æ `^`, `$`, `*`, `+` —ñ `?` –Ω–µ –∑–∞—Ä–µ–∑–µ—Ä–≤–æ–≤–∞–Ω—ñ –≤ –º–µ–∂–∞—Ö –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤, –∞ `/` —ñ `-` –Ω–µ –∑–∞—Ä–µ–∑–µ—Ä–≤–æ–≤–∞–Ω—ñ –ø–æ–∑–∞ –º–µ–∂–∞–º–∏ –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤ (—Ö–æ—á–∞ `/` –º–æ–∂–µ –≤—ñ–¥–º–µ–∂–æ–≤—É–≤–∞—Ç–∏ –ª—ñ—Ç–µ—Ä–∞–ª —Ä–µ–≥—É–ª—è—Ä–Ω–æ–≥–æ –≤–∏—Ä–∞–∑—É —ñ —Ç–æ–º—É –≤—Å–µ —â–µ –ø–æ—Ç—Ä–µ–±—É—î –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è). –£—Å—ñ —Ü—ñ —Å–∏–º–≤–æ–ª–∏ —Ç–∞–∫–æ–∂ –º–æ–∂—É—Ç—å –±—É—Ç–∏ –µ–∫—Ä–∞–Ω–æ–≤–∞–Ω—ñ –∑–∞ –±–∞–∂–∞–Ω–Ω—è–º –≤ –∫–ª–∞—Å–∞—Ö —Å–∏–º–≤–æ–ª—ñ–≤ —É —Ä–µ–∂–∏–º—ñ `u`.
- –ù–∞—Å—Ç—É–ø–Ω—ñ –ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ "–ø–æ–¥–≤—ñ–π–Ω–∏—Ö —Ä–æ–∑–¥—ñ–ª–æ–≤–∏—Ö" —Ç–∞–∫–æ–∂ –ø–æ–≤–∏–Ω–Ω—ñ –±—É—Ç–∏ –µ–∫—Ä–∞–Ω–æ–≤–∞–Ω—ñ (–∞–ª–µ –≤–æ–Ω–∏ –≤—Å–µ –æ–¥–Ω–æ –º–∞—é—Ç—å –Ω–µ–±–∞–≥–∞—Ç–æ –∑–º—ñ—Å—Ç—É –±–µ–∑ –ø–æ–∑–Ω–∞—á–∫–∏ `v`): `&&`, `!!`, `##`, `$$`, `%%`, `**`, `++`, `,,`, `..`, `::`, `;;`, `<<`, `==`, `>>`, `??`, `@@`, `^^`, ` `` `, `~~`. –£ —Ä–µ–∂–∏–º—ñ `u` –¥–µ—è–∫—ñ –∑ —Ü–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤ –º–æ–∂—É—Ç—å –∑'—è–≤–ª—è—Ç–∏—Å—è –±—É–∫–≤–∞–ª—å–Ω–æ –ª–∏—à–µ –≤ –º–µ–∂–∞—Ö –∫–ª–∞—Å—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤ —ñ –≤–∏–∫–ª–∏–∫–∞—é—Ç—å —Å–∏–Ω—Ç–∞–∫—Å–∏—á–Ω—É –ø–æ–º–∏–ª–∫—É –ø—Ä–∏ —Å–ø—Ä–æ–±—ñ —ó—Ö –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏. –£ —Ä–µ–∂–∏–º—ñ `v` –≤–æ–Ω–∏ –ø–æ–≤–∏–Ω–Ω—ñ –±—É—Ç–∏ –µ–∫—Ä–∞–Ω–æ–≤–∞–Ω—ñ, –∫–æ–ª–∏ –∑'—è–≤–ª—è—é—Ç—å—Å—è –ø–∞—Ä–∞–º–∏, –∞–ª–µ –º–æ–∂—É—Ç—å –±—É—Ç–∏ –µ–∫—Ä–∞–Ω–æ–≤–∞–Ω—ñ –∑–∞ –±–∞–∂–∞–Ω–Ω—è–º, –∫–æ–ª–∏ –∑'—è–≤–ª—è—é—Ç—å—Å—è –æ–∫—Ä–µ–º–æ. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `/[\!]/u` —î –Ω–µ–¥—ñ–π—Å–Ω–∏–º –∑–∞–ø–∏—Å–æ–º, –æ—Å–∫—ñ–ª—å–∫–∏ —Ü–µ [–µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è —ñ–¥–µ–Ω—Ç–∏—á–Ω–æ—Å—Ç—ñ](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Character_escape), –∞–ª–µ —è–∫ `/[\!]/v`, —Ç–∞–∫ —ñ `/[!]/v` —î –¥—ñ–π—Å–Ω–∏–º–∏, –∞–ª–µ `/[!!]/v` —î –Ω–µ–¥—ñ–π—Å–Ω–∏–º. –î–æ–≤—ñ–¥–∫–∞ [–±—É–∫–≤–∞–ª—å–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤](/uk/docs/Web/JavaScript/Reference/Regular_expressions/Literal_character) –º—ñ—Å—Ç–∏—Ç—å –¥–æ–∫–ª–∞–¥–Ω—É —Ç–∞–±–ª–∏—Ü—é —Ç–æ–≥–æ, —è–∫—ñ —Å–∏–º–≤–æ–ª–∏ –º–æ–∂—É—Ç—å –∑—É—Å—Ç—Ä—ñ—á–∞—Ç–∏—Å—è –∑ –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è–º, —ñ —è–∫—ñ –±–µ–∑ –Ω—å–æ–≥–æ.

–î–æ–ø–æ–≤–Ω—è–ª—å–Ω—ñ –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ `[^...]` –≤–∑–∞–≥–∞–ª—ñ –Ω–µ –º–æ–∂—É—Ç—å –∑–±—ñ–≥–∞—Ç–∏—Å—è –∑ —Ä—è–¥–∫–∞–º–∏, –¥–æ–≤—à–∏–º–∏ –∑–∞ –æ–¥–∏–Ω —Å–∏–º–≤–æ–ª. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, `[\q{ab|c}]` —î –¥—ñ–π—Å–Ω–∏–º —ñ –¥–∞—î –∑–±—ñ–≥ –∑ —Ä—è–¥–∫–æ–º `"ab"`, –∞–ª–µ `[^\q{ab|c}]` —î –Ω–µ–¥—ñ–π—Å–Ω–∏–º, –æ—Å–∫—ñ–ª—å–∫–∏ –Ω–µ—è—Å–Ω–æ, —Å–∫—ñ–ª—å–∫–∏ —Å–∏–º–≤–æ–ª—ñ–≤ –ø–æ—Ç—Ä—ñ–±–Ω–æ —Å–ø–æ–∂–∏—Ç–∏. –ü–µ—Ä–µ–≤—ñ—Ä–∫–∞ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è —à–ª—è—Ö–æ–º –ø–µ—Ä–µ–≤—ñ—Ä–∫–∏ —Ç–æ–≥–æ, —á–∏ –≤—Å—ñ `\q` –º—ñ—Å—Ç—è—Ç—å –æ–¥–∏–Ω —Å–∏–º–≤–æ–ª, —ñ —á–∏ –≤—Å—ñ `\p` –≤–∫–∞–∑—É—é—Ç—å –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ —Å–∏–º–≤–æ–ª—ñ–≤ ‚Äî –¥–ª—è –æ–±'—î–¥–Ω–∞–Ω—å –≤—Å—ñ –æ–ø–µ—Ä–∞–Ω–¥–∏ –ø–æ–≤–∏–Ω–Ω—ñ –±—É—Ç–∏ —á–∏—Å—Ç–æ —Å–∏–º–≤–æ–ª–∞–º–∏; –¥–ª—è –ø–µ—Ä–µ—Ç–∏–Ω—ñ–≤, –ø—Ä–∏–Ω–∞–π–º–Ω—ñ –æ–¥–∏–Ω –æ–ø–µ—Ä–∞–Ω–¥ –ø–æ–≤–∏–Ω–µ–Ω –±—É—Ç–∏ —á–∏—Å—Ç–æ —Å–∏–º–≤–æ–ª–æ–º; –¥–ª—è –≤—ñ–¥–Ω—ñ–º–∞–Ω–Ω—è, –ª—ñ–≤–∏–π –æ–ø–µ—Ä–∞–Ω–¥ –ø–æ–≤–∏–Ω–µ–Ω –±—É—Ç–∏ —á–∏—Å—Ç–æ —Å–∏–º–≤–æ–ª–æ–º. –ü–µ—Ä–µ–≤—ñ—Ä–∫–∞ —î —Å–∏–Ω—Ç–∞–∫—Å–∏—á–Ω–æ—é –±–µ–∑ –ø–µ—Ä–µ–≥–ª—è–¥—É —Ñ–∞–∫—Ç–∏—á–Ω–æ–≥–æ –Ω–∞–±–æ—Ä—É —Å–∏–º–≤–æ–ª—ñ–≤, —è–∫–∏–π –≤–∫–∞–∑—É—î—Ç—å—Å—è, —â–æ –æ–∑–Ω–∞—á–∞—î, —â–æ —Ö–æ—á–∞ `/[^\q{ab|c}--\q{ab}]/v` –µ–∫–≤—ñ–≤–∞–ª–µ–Ω—Ç–Ω–∏–π `/[^c]/v`, –≤—ñ–Ω –≤—Å–µ —â–µ –≤—ñ–¥—Ö–∏–ª—è—î—Ç—å—Å—è.

### –î–æ–ø–æ–≤–Ω—è–ª—å–Ω—ñ –∫–ª–∞—Å–∏ —Ç–∞ —Ä–µ–∂–∏–º —ñ–≥–Ω–æ—Ä—É–≤–∞–Ω–Ω—è —Ä–µ–≥—ñ—Å—Ç—Ä—É

–£ —Ä–µ–∂–∏–º—ñ –±–µ–∑ `v` –¥–æ–ø–æ–≤–Ω—è–ª—å–Ω—ñ –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ `[^...]` —Ä–µ–∞–ª—ñ–∑–æ–≤–∞–Ω—ñ –ø—Ä–æ—Å—Ç–æ —à–ª—è—Ö–æ–º –æ–±–µ—Ä—Ç–∞–Ω–Ω—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É –∑–±—ñ–≥—É ‚Äì —Ç–æ–±—Ç–æ `[^...]` –¥–∞—î –∑–±—ñ–≥ —â–æ—Ä–∞–∑—É, –∫–æ–ª–∏ `[...]` –∑–±—ñ–≥—É –Ω–µ –¥–∞—î, —ñ –Ω–∞–≤–ø–∞–∫–∏. –ü—Ä–æ—Ç–µ —ñ–Ω—à—ñ –¥–æ–ø–æ–≤–Ω—è–ª—å–Ω—ñ –∫–ª–∞—Å–∏, —è–∫ —Ç–æ `\P{...}` —ñ `\W`, –ø—Ä–∞—Ü—é—é—Ç—å —à–ª—è—Ö–æ–º –Ω–µ–≥–∞–π–Ω–æ—ó –ø–æ–±—É–¥–æ–≤–∏ –Ω–∞–±–æ—Ä—É, —â–æ —Å–∫–ª–∞–¥–∞—î—Ç—å—Å—è –∑ —É—Å—ñ—Ö —Å–∏–º–≤–æ–ª—ñ–≤ –±–µ–∑ –≤–∫–∞–∑–∞–Ω–æ—ó –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ. –í–æ–Ω–∏, –∑–¥–∞—î—Ç—å—Å—è, –¥–∞—é—Ç—å —Ç–æ–π —Å–∞–º–∏–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç, –∞–ª–µ —Å—Ç–∞—é—Ç—å —Å–∫–ª–∞–¥–Ω—ñ—à–∏–º–∏ –ø—Ä–∏ –ø–æ—î–¥–Ω–∞–Ω–Ω—ñ –∑ [—ñ–≥–Ω–æ—Ä—É–≤–∞–Ω–Ω—è–º —Ä–µ–≥—ñ—Å—Ç—Ä—É](/uk/docs/Web/JavaScript/Reference/Global_Objects/RegExp/ignoreCase).

–î–ª—è –ø—Ä–∏–∫–ª–∞–¥—É ‚Äì —Ç–∞–∫—ñ –¥–≤–∞ —Ä–µ–≥—É–ª—è—Ä–Ω—ñ –≤–∏—Ä–∞–∑–∏:

```js
const r1 = /\p{Lowercase_Letter}/iu;
const r2 = /[^\P{Lowercase_Letter}]/iu;
```

–í–∏—Ä–∞–∑ `r2` ‚Äì —Ü–µ –ø–æ–¥–≤—ñ–π–Ω–µ –∑–∞–ø–µ—Ä–µ—á–µ–Ω–Ω—è —ñ, –∑–¥–∞—î—Ç—å—Å—è, —Ä—ñ–≤–Ω–æ—Å–∏–ª—å–Ω–∏–π `r1`. –ê–ª–µ –Ω–∞—Å–ø—Ä–∞–≤–¥—ñ `r1` –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ —É—Å—ñ–º–∞ —Å–∏–º–≤–æ–ª–∞–º–∏ –Ω–∏–∂–Ω—å–æ–≥–æ —ñ –≤–µ—Ä—Ö–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—ñ–≤ ASCII, —Ç–æ–¥—ñ —è–∫ `r2` –Ω–µ –∑–±—ñ–≥–∞—î—Ç—å—Å—è –∑ –∂–æ–¥–Ω–∏–º. –©–æ–± –ø—Ä–æ—ñ–ª—é—Å—Ç—Ä—É–≤–∞—Ç–∏, —è–∫ —Ü–µ –ø—Ä–∞—Ü—é—î, —É–¥–∞–π–º–æ, —â–æ –º–∞—î–º–æ —Å–ø—Ä–∞–≤—É –ª–∏—à–µ –∑ —Å–∏–º–≤–æ–ª–∞–º–∏ ASCII, –∞ –Ω–µ –∑ —É—Å—ñ–º –Ω–∞–±–æ—Ä–æ–º —Å–∏–º–≤–æ–ª—ñ–≤ Unicode, —ñ `r1` —ñ `r2` –∑–∞–¥–∞–Ω—ñ —Ç–∞–∫:

```js
const r1 = /[a-z]/iu;
const r2 = /[^A-Z]/iu;
```

–ü—Ä–∏–≥–∞–¥–∞–π—Ç–µ, —â–æ –ø–æ—à—É–∫ –∑–±—ñ–≥—É –∑ —ñ–≥–Ω–æ—Ä—É–≤–∞–Ω–Ω—è–º —Ä–µ–≥—ñ—Å—Ç—Ä—É –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è —à–ª—è—Ö–æ–º –∑–≤–µ–¥–µ–Ω–Ω—è —ñ –ø–∞—Ç–µ—Ä–Ω—É, —ñ –≤–∏—Ö—ñ–¥–Ω–æ–≥–æ —Ä—è–¥–∫—É –¥–æ –æ–¥–Ω–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É (–¥–æ–∫–ª–∞–¥–Ω—ñ—à–µ –≤ {{jsxref("RegExp/ignoreCase", "ignoreCase")}}). –î–ª—è `r1` –∫–ª–∞—Å —Å–∏–º–≤–æ–ª—ñ–≤ `a-z` –∑–∞–ª–∏—à–∞—î—Ç—å—Å—è —Ç–∞–∫–∏–º —Å–∞–º–∏–º –ø—ñ—Å–ª—è –∑–≤–µ–¥–µ–Ω–Ω—è —Ä–µ–≥—ñ—Å—Ç—Ä—É, —Ç–æ–¥—ñ —è–∫ –æ–±–∏–¥–≤–∞ —Ä—è–¥–∫–∏ ASCII –≤–µ—Ä—Ö–Ω—å–æ–≥–æ —ñ –Ω–∏–∂–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É –∑–≤–æ–¥—è—Ç—å—Å—è –¥–æ –Ω–∏–∂–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É, —Ç–æ–º—É `r1` –º–æ–∂–µ –∑–±—ñ–≥–∞—Ç–∏—Å—è —è–∫ —ñ–∑ `"A"`, —Ç–∞–∫ —ñ –∑ `"a"`. –î–ª—è `r2` –∫–ª–∞—Å —Å–∏–º–≤–æ–ª—ñ–≤ `A-Z` –∑–≤–æ–¥–∏—Ç—å—Å—è –¥–æ `a-z`; –ø—Ä–æ—Ç–µ `^` –∑–∞–ø–µ—Ä–µ—á—É—î —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∑–±—ñ–≥—É, —Ç–∞–∫ —â–æ `[^A-Z]` –Ω–∞—Å–ø—Ä–∞–≤–¥—ñ –∑–±—ñ–≥–∞—î—Ç—å—Å—è –ª–∏—à–µ –∑ —Ä—è–¥–∫–∞–º–∏ –≤–µ—Ä—Ö–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É. –û–¥–Ω–∞–∫ —ñ —Ä—è–¥–∫–∏ ASCII –≤–µ—Ä—Ö–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É, —ñ —Ä—è–¥–∫–∏ –Ω–∏–∂–Ω—å–æ–≥–æ ‚Äì –æ–¥–Ω–∞–∫–æ–≤–æ –∑–≤–æ–¥—è—Ç—å—Å—è –¥–æ –Ω–∏–∂–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É, —â–æ –ø—Ä–∏–∑–≤–æ–¥–∏—Ç—å –¥–æ —Ç–æ–≥–æ, —â–æ `r2` –Ω–µ –∑–±—ñ–≥–∞—î—Ç—å—Å—è –Ω—ñ –∑ —á–∏–º.

–£ —Ä–µ–∂–∏–º—ñ `v` —Ü—è –ª–æ–≥—ñ–∫–∞ –≤–∏–ø—Ä–∞–≤–ª–µ–Ω–∞: `[^...]` —Ç–∞–∫–æ–∂ –Ω–µ–≥–∞–π–Ω–æ –≤–∏–±—É–¥–æ–≤—É—î –¥–æ–ø–æ–≤–Ω—è–ª—å–Ω–∏–π –∫–ª–∞—Å, –∞ –Ω–µ –∑–∞–ø–µ—Ä–µ—á—É—î —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∑–±—ñ–≥—É. –¶–µ —Ä–æ–±–∏—Ç—å `[^\P{Lowercase_Letter}]` —ñ `\p{Lowercase_Letter}` —Å—Ç—Ä–æ–≥–æ —Ä—ñ–≤–Ω–æ—Å–∏–ª—å–Ω–∏–º–∏.

## –ü—Ä–∏–∫–ª–∞–¥–∏

### –ü–æ—à—É–∫ —à—ñ—Å—Ç–Ω–∞–¥—Ü—è—Ç–∫–æ–≤–∏—Ö —Ü–∏—Ñ—Ä

–ù–∞—Å—Ç—É–ø–Ω–∞ —Ñ—É–Ω–∫—Ü—ñ—è –∑'—è—Å–æ–≤—É—î, —á–∏ –º—ñ—Å—Ç–∏—Ç—å —Ä—è–¥–æ–∫ –¥—ñ–π—Å–Ω–µ —à—ñ—Å—Ç–Ω–∞–¥—Ü—è—Ç–∫–æ–≤–µ —á–∏—Å–ª–æ:

```js

  return /^[0-9A-F]+$/i.test(str);
}

isHexadecimal("2F3"); // true
isHexadecimal("beef"); // true
isHexadecimal("undefined"); // false
```

### –ó–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è –ø–µ—Ä–µ—Ç–∏–Ω—É

–ù–∞—Å—Ç—É–ø–Ω–∞ —Ñ—É–Ω–∫—Ü—ñ—è —à—É–∫–∞—î –∑–±—ñ–≥–∏ –∑ –≥—Ä–µ—Ü—å–∫–∏–º–∏ –ª—ñ—Ç–µ—Ä–∞–º–∏.

```js
function greekLetters(str) {
  return str.match(/[\p{Script_Extensions=Greek}&&\p{Letter}]/gv);

```

### –ó–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è —Ä—ñ–∑–Ω–∏—Ü—ñ

–ù–∞—Å—Ç—É–ø–Ω–∞ —Ñ—É–Ω–∫—Ü—ñ—è —à—É–∫–∞—î –≤—Å—ñ —Ü–∏—Ñ—Ä–∏, —è–∫—ñ –Ω–µ —î —Å–∏–º–≤–æ–ª–∞–º–∏ ASCII.

```js

// ëúπ ‚Äì —Ü–µ U+11739 AHOM DIGIT NINE (–∞—Ö–æ–º—Å—å–∫–∞ —Ü–∏—Ñ—Ä–∞ –¥–µ–≤'—è—Ç—å)
```
