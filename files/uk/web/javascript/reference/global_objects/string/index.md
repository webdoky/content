### –î–æ–≤–≥—ñ —Ä—è–¥–∫–∏ –∑ –ª—ñ—Ç–µ—Ä

–Ü–Ω–∫–æ–ª–∏ –∫–æ–¥ –º–æ–∂–µ –º—ñ—Å—Ç–∏—Ç–∏ –¥—É–∂–µ –¥–æ–≤–≥—ñ —Ä—è–¥–∫–∏. –á—Ö –º–æ–∂–Ω–∞ —Ä–æ–∑–±–∏—Ç–∏ –Ω–∞ –∫—ñ–ª—å–∫–∞ —Ä—è–¥–∫—ñ–≤ —É –≤–∏—Ö—ñ–¥–Ω–æ–º—É –∫–æ–¥—ñ, –Ω–µ –∑–∞—à–∫–æ–¥–∏–≤—à–∏ –≤–º—ñ—Å—Ç—É –∞–±–æ —Ä—è–¥–∫–∞–º. –¶–µ –¥–∞—î –∑–º–æ–≥—É –Ω–µ –∑–∞–ª–∏—à–∞—Ç–∏ –Ω–µ—Å–∫—ñ–Ω—á–µ–Ω–Ω—ñ —Ä—è–¥–∫–∏ —Ç–∞ –Ω–µ –ø–æ–∫–ª–∞–¥–∞—Ç–∏ –ø–µ—Ä–µ–Ω–µ—Å–µ–Ω–Ω—è —Ä—è–¥–∫—ñ–≤ –Ω–∞ —Ç–µ–∫—Å—Ç–æ–≤–∏–π —Ä–µ–¥–∞–∫—Ç–æ—Ä. –Ü—Å–Ω—É—é—Ç—å –¥–≤–∞ —Å–ø–æ—Å–æ–±–∏ –∑—Ä–æ–±–∏—Ç–∏ —Ü–µ.

–ú–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä [`+`](/uk/docs/Web/JavaScript/Reference/Operators/Addition) –¥–ª—è –∑'—î–¥–Ω–∞–Ω–Ω—è –¥–æ–∫—É–ø–∏ –¥–µ–∫—ñ–ª—å–∫–æ—Ö —Ä—è–¥–∫—ñ–≤, —è–∫ —Ü–µ –ø–æ–∫–∞–∑–∞–Ω–æ –Ω–∏–∂—á–µ:

```js
const longString =
  '–¶–µ ‚Äî –¥—É–∂–µ –¥–æ–≤–≥–∞ —Å—Ç—Ä—ñ—á–∫–∞ —Ç–µ–∫—Å—Ç—É, —è–∫—É –ø–æ—Ç—Ä—ñ–±–Ω–æ ' +
  '—Ä–æ–∑–±–∏—Ç–∏ –Ω–∞ –¥–µ–∫—ñ–ª—å–∫–∞ –æ–∫—Ä–µ–º–∏—Ö —Ä—è–¥–∫—ñ–≤, –±–æ ' +
  '—ñ–Ω–∞–∫—à–µ —Ü–µ–π –∫–æ–¥ –±—É–¥–µ –≤–∞–∂–∫–æ —á–∏—Ç–∞—Ç–∏.';
```

–ê–±–æ –∂ –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ –æ–±–µ—Ä–Ω–µ–Ω—É —Å–∫—ñ—Å–Ω—É —Ä–∏—Å–∫—É (`\`) –≤ –∫—ñ–Ω—Ü—ñ –∫–æ–∂–Ω–æ–≥–æ —Ä—è–¥–∫–∞, —â–æ–± –ø–æ–∑–Ω–∞—á–∏—Ç–∏, —â–æ —Å—Ç—Ä—ñ—á–∫–∞ –ø—Ä–æ–¥–æ–≤–∂—É—î—Ç—å—Å—è –Ω–∞ –Ω–∞—Å—Ç—É–ø–Ω–æ–º—É —Ä—è–¥–∫—É. –û–±–æ–≤'—è–∑–∫–æ–≤–æ —Å–ª—ñ–¥ –≤–ø–µ–≤–Ω–∏—Ç–∏—Å—å, —â–æ –ø—ñ—Å–ª—è —Å–∫–æ—Å—É –Ω–µ–º–∞—î –ø—Ä–æ–±—ñ–ª—É —á–∏ —è–∫–∏—Ö–æ—Å—å —ñ–Ω—à–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤ (–æ–∫—Ä—ñ–º –ø–µ—Ä–µ–Ω–µ—Å–µ–Ω–Ω—è –Ω–∞ –Ω–æ–≤–∏–π —Ä—è–¥–æ–∫), –∞–±–æ –≤—ñ–¥—Å—Ç—É–ø—ñ–≤, —ñ–Ω–∞–∫—à–µ —Ç–∞–∫–∏–π –∑–∞–ø–∏—Å –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ –Ω–µ –±—É–¥–µ.

–£ —Ü—ñ–π —Ñ–æ—Ä–º—ñ —Ä—è–¥–æ–∫ –º–∞—Ç–∏–º–µ —Ç–∞–∫–∏–π –≤–∏–≥–ª—è–¥:


–û–±–∏–¥–≤–∞ –º–µ—Ç–æ–¥–∏ –¥–∞–¥—É—Ç—å –≤ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ —ñ–¥–µ–Ω—Ç–∏—á–Ω—ñ —Ä—è–¥–∫–∏.

### –°–∏–º–≤–æ–ª–∏ UTF-16, –∫–æ–¥–æ–≤—ñ —Ç–æ—á–∫–∏ –Æ–Ω—ñ–∫–æ–¥—É —Ç–∞ –≥—Ä–∞—Ñ–µ–º–Ω—ñ –∫–ª–∞—Å—Ç–µ—Ä–∏

–†—è–¥–∫–∏ —É —Å–≤–æ—ó–π –æ—Å–Ω–æ–≤—ñ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ñ —è–∫ –ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ [–∫–æ–¥–æ–≤–∏—Ö —Ç–æ—á–æ–∫ UTF-16](https://uk.wikipedia.org/wiki/UTF-16). –£ –∫–æ–¥—É–≤–∞–Ω–Ω—ñ UTF-16 –∫–æ–∂–Ω–∞ –∫–æ–¥–æ–≤–∞ —Ç–æ—á–∫–∞ ‚Äì —Ä—ñ–≤–Ω–æ 16 –±—ñ—Ç—ñ–≤ –∑–∞–≤–¥–æ–≤–∂–∫–∏. –¶–µ –æ–∑–Ω–∞—á–∞—î, —â–æ —î —â–æ–Ω–∞–π–±—ñ–ª—å—à–µ 2<sup>16</sup>, —Ç–æ–±—Ç–æ 65536, –º–æ–∂–ª–∏–≤–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤, –∫–æ—Ç—Ä—ñ –º–æ–∂–Ω–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç–∏ —É –≤–∏–≥–ª—è–¥—ñ –∫–æ–¥–æ–≤–æ—ó —Ç–æ—á–∫–∏ UTF-16. –¶–µ–π –Ω–∞–±—ñ—Ä —Å–∏–º–≤–æ–ª—ñ–≤ –∑–≤–µ—Ç—å—Å—è [–±–∞–≥–∞—Ç–æ–º–æ–≤–Ω–æ—é –ø–ª–æ—â–∏–Ω–æ—é 0](<https://uk.wikipedia.org/wiki/%D0%91%D0%B0%D0%B3%D0%B0%D1%82%D0%BE%D0%BC%D0%BE%D0%B2%D0%BD%D0%B0_%D0%BF%D0%BB%D0%BE%D1%89%D0%B8%D0%BD%D0%B0_(%D0%AE%D0%BD%D1%96%D0%BA%D0%BE%D0%B4)>) (—Ç–∞–∫–æ–∂ BMP) —ñ –º—ñ—Å—Ç–∏—Ç—å –±—ñ–ª—å—à—ñ—Å—Ç—å –ø–æ—à–∏—Ä–µ–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤ —Ç–∏–ø—É –ª–∞—Ç–∏–Ω—Å—å–∫–æ—ó, –≥—Ä–µ—Ü—å–∫–æ—ó, –∫–∏—Ä–∏–ª–∏—á–Ω–æ—ó –∞–±–µ—Ç–æ–∫, –∞ —Ç–∞–∫–æ–∂ –±–∞–≥–∞—Ç—å–æ—Ö —Å—Ö—ñ–¥–Ω–æ–∞–∑—ñ–π—Å—å–∫–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤. –ö–æ–∂–Ω–∞ –∫–æ–¥–æ–≤–∞ —Ç–æ—á–∫–∞ –º–æ–∂–µ –±—É—Ç–∏ –∑–∞–ø–∏—Å–∞–Ω–∞ —è–∫ —Ä—è–¥–æ–∫, –¥–µ –Ω–∞ –ø–æ—á–∞—Ç–∫—É —Å—Ç–æ—ó—Ç—å `\u`, –∞ –¥–∞–ª—ñ ‚Äì —Ä—ñ–≤–Ω–æ —á–æ—Ç–∏—Ä–∏ —à—ñ—Å—Ç–Ω–∞–¥—Ü—è—Ç–∫–æ–≤—ñ —Ü–∏—Ñ—Ä–∏.

–ü—Ä–æ—Ç–µ —É–≤–µ—Å—å –Ω–∞–±—ñ—Ä —Å–∏–º–≤–æ–ª—ñ–≤ –Æ–Ω—ñ–∫–æ–¥—É —î –Ω–∞–±–∞–≥–∞—Ç–æ, –Ω–∞–±–∞–≥–∞—Ç–æ –±—ñ–ª—å—à–∏–º –∑–∞ 65536. –î–æ–¥–∞—Ç–∫–æ–≤—ñ —Å–∏–º–≤–æ–ª–∏ –∑–±–µ—Ä—ñ–≥–∞—é—Ç—å—Å—è –≤ UTF-16 —è–∫ _—Å—É—Ä–æ–≥–∞—Ç–Ω—ñ –ø–∞—Ä–∏_, –∫–æ—Ç—Ä—ñ —î –ø–∞—Ä–∞–º–∏ 16-–±—ñ—Ç–æ–≤–∏—Ö –∫–æ–¥–æ–≤–∏—Ö –æ–¥–∏–Ω–∏—Ü—å, –∫–æ–∂–Ω–∞ –∑ —è–∫–∏—Ö –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—î —î–¥–∏–Ω–∏–π —Å–∏–º–≤–æ–ª. –î–ª—è —É–Ω–∏–∫–Ω–µ–Ω–Ω—è –Ω–µ–æ–¥–Ω–æ–∑–Ω–∞—á–Ω–æ—Å—Ç—ñ –¥–≤—ñ —á–∞—Å—Ç–∏–Ω–∏ –ø–∞—Ä–∏ –º—É—Å—è—Ç—å –ª–µ–∂–∞—Ç–∏ –º—ñ–∂ `0xD800` —Ç–∞ `0xDFFF`, —ñ —Ü—ñ –∫–æ–¥–æ–≤—ñ –æ–¥–∏–Ω–∏—Ü—ñ –Ω–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –∫–æ–¥—É–≤–∞–Ω–Ω—è —Å–∏–º–≤–æ–ª—ñ–≤ –∑ –æ–¥–Ω—ñ—î—ó –∫–æ–¥–æ–≤–æ—ó –æ–¥–∏–Ω–∏—Ü—ñ. –¢–∞–∫–∏–º —á–∏–Ω–æ–º, "—Å–∞–º–æ—Ç–Ω—ñ —Å—É—Ä–æ–≥–∞—Ç–∏" —á–∞—Å—Ç–æ –Ω–µ —î –¥—ñ–π—Å–Ω–∏–º–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º–∏ –¥–ª—è –ø–æ–≤–æ–¥–∂–µ–Ω–Ω—è –∑ —Ä—è–¥–∫–∞–º–∏, ‚Äì –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, [`encodeURI()`](/uk/docs/Web/JavaScript/Reference/Global_Objects/encodeURI) –≤–∏–∫–∏–Ω–µ {{jsxref("URIError")}}, –∑—É—Å—Ç—Ä—ñ–≤—à–∏ —Å–∞–º–æ—Ç–Ω—ñ–π —Å—É—Ä–æ–≥–∞—Ç. –ö–æ–∂–µ–Ω —Å–∏–º–≤–æ–ª –Æ–Ω—ñ–∫–æ–¥—É, —â–æ —Å–∫–ª–∞–¥–∞—î—Ç—å—Å—è –∑ –æ–¥–Ω—ñ—î—ó —á–∏ –¥–≤–æ—Ö –∫–æ–¥–æ–≤–∏—Ö –æ–¥–∏–Ω–∏—Ü—å UTF-16, —Ç–∞–∫–æ–∂ –∑–≤–µ—Ç—å—Å—è _–∫–æ–¥–æ–≤–æ—é —Ç–æ—á–∫–æ—é –Æ–Ω—ñ–∫–æ–¥—É_. –ö–æ–∂–Ω–∞ –∫–æ–¥–æ–≤–∞ —Ç–æ—á–∫–∞ –Æ–Ω—ñ–∫–æ–¥—É –º–æ–∂–µ –±—É—Ç–∏ –∑–∞–ø–∏—Å–∞–Ω–∞ –≤ —Ä—è–¥–æ–∫ —è–∫ `\u{xxxxxx}`, –¥–µ `xxxxxx` ‚Äì –≤—ñ–¥ –æ–¥–Ω—ñ—î—ó –¥–æ 6 —à—ñ—Å—Ç–Ω–∞–¥—Ü—è—Ç–∫–æ–≤–∏—Ö —Ü–∏—Ñ—Ä.

–ù–∞ –¥–æ–¥–∞—á—É –¥–æ —Å–∏–º–≤–æ–ª—ñ–≤ –Æ–Ω—ñ–∫–æ–¥—É, —î –ø–µ–≤–Ω—ñ –ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ —Å–∏–º–≤–æ–ª—ñ–≤ –Æ–Ω—ñ–∫–æ–¥—É, –∫–æ—Ç—Ä—ñ –ø–æ–≤–∏–Ω–Ω—ñ —Ä–æ–∑–≥–ª—è–¥–∞—Ç–∏—Å—è —è–∫ —î–¥–∏–Ω–∞ –≤–∏–¥–∏–º–∞ –æ–¥–∏–Ω–∏—Ü—è, ‚Äì –≤–æ–Ω–∏ –≤—ñ–¥–æ–º—ñ —è–∫ _–≥—Ä–∞—Ñ–µ–º–Ω—ñ –∫–ª–∞—Å—Ç–µ—Ä–∏_. –ù–∞–π–ø–æ—à–∏—Ä–µ–Ω—ñ—à–∏–π –ø—Ä–∏–∫–ª–∞–¥ ‚Äì –µ–º–æ–¥–∂—ñ: —á–∏–º–∞–ª–æ –µ–º–æ–¥–∂—ñ, —â–æ –º–∞—é—Ç—å –∑–Ω–∞—á–Ω–∏–π –¥—ñ–∞–ø–∞–∑–æ–Ω –≤–∞—Ä—ñ–∞—Ü—ñ–π, –Ω–∞—Å–ø—Ä–∞–≤–¥—ñ —É—Ç–≤–æ—Ä—é—é—Ç—å—Å—è –¥–µ–∫—ñ–ª—å–∫–æ–º–∞ –µ–º–æ–¥–∑—ñ, –∑–∞–∑–≤–∏—á–∞–π —Å–ø–æ–ª—É—á–µ–Ω–∏–º–∏ —Å–∏–º–≤–æ–ª–æ–º \<ZWJ> (`U+200D`).

–°–ª—ñ–¥ –±—É—Ç–∏ –æ–±–µ—Ä–µ–∂–Ω–∏–º–∏ –∑ —Ä—ñ–≤–Ω—è–º–∏ —Å–∏–º–≤–æ–ª—ñ–≤, –Ω–∞ –∫–æ—Ç—Ä–∏—Ö –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è –æ–±—Ö—ñ–¥. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, [`split("")`](/uk/docs/Web/JavaScript/Reference/Global_Objects/String/split) —Ä–æ–∑—ñ–±'—î –Ω–∞ –∫–æ–¥–æ–≤—ñ –æ–¥–∏–Ω–∏—Ü—ñ UTF-16 —ñ —Ä–æ–∑–¥—ñ–ª–∏—Ç—å —Å—É—Ä–æ–≥–∞—Ç–Ω—ñ –ø–∞—Ä–∏. –†—è–¥–∫–æ–≤—ñ —ñ–Ω–¥–µ–∫—Å–∏ —Ç–∞–∫–æ–∂ –≤–∫–∞–∑—É—é—Ç—å –Ω–∞ —ñ–Ω–¥–µ–∫—Å–∏ –≤—Å—ñ—Ö –∫–æ–¥–æ–≤–∏—Ö –æ–¥–∏–Ω–∏—Ü—å UTF-16. –ó —ñ–Ω—à–æ–≥–æ –±–æ–∫—É, [`@@iterator()`](/uk/docs/Web/JavaScript/Reference/Global_Objects/String/@@iterator) –≤–∏–∫–æ–Ω—É—î –æ–±—Ö—ñ–¥ –∑–∞ –∫–æ–¥–æ–≤–∏–º–∏ —Ç–æ—á–∫–∞–º–∏ –Æ–Ω—ñ–∫–æ–¥—É. –û–±—Ö—ñ–¥ –∑–∞ –≥—Ä–∞—Ñ–µ–º–Ω–∏–º–∏ –∫–ª–∞—Å—Ç–µ—Ä–∞–º–∏ –ø–æ—Ç—Ä–µ–±—É–≤–∞—Ç–∏–º–µ –Ω–∞–ø–∏—Å–∞–Ω–Ω—è –ø–µ–≤–Ω–æ–≥–æ –∫–æ–¥—É —Å–∞–º–æ—Ç—É–∂–∫–∏.
