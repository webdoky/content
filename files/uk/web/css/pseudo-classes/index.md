[CSS](/uk/docs/Web/CSS) **_псевдоклас_** — це ключове слово, яке додають до селектора, щоб стилізувати певний стан елемента(ів). Наприклад, псевдоклас {{CSSxRef(":hover")}} дає змогу вибрати кнопку під час наведення вказівника та застосувати потрібні стилі.

```css
/* Будь-яка кнопка, над якою знаходиться вказівник користувача */
button:hover {
  color: blue;
}
```

Псевдоклас починається двокрапкою (`:`), далі йде його назва (наприклад, `:hover`). Для функціонального псевдокласу використовують круглі дужки, щоб передати аргументи (наприклад, `:dir()`). Елемент із псевдокласом називається _якірним_ (наприклад, `button` у `button:hover`).

Псевдокласи допомагають стилізувати елемент не лише залежно від вмісту документа, а й з урахуванням зовнішніх чинників: історії браузера ({{CSSxRef(":visited")}}), стану вмісту (наприклад, {{CSSxRef(":checked")}}) чи положення миші ({{CSSxRef(":hover")}}).

> [!NOTE]
> На відміну від псевдокласів, [псевдоелементи](/uk/docs/Web/CSS/Pseudo-elements) можна використовувати для стилізації _конкретної частини_ елемента.

## Псевдокласи стану відображення елемента

Ці псевдокласи дозволяють вибирати елементи на основі їх стану відображення.

- {{CSSxRef(":fullscreen")}}
  - : Відповідає елементу, який наразі знаходиться в режимі повноекранного відображення.
- {{CSSxRef(":modal")}}
  - : Відповідає елементу, який перебуває у стані, коли він виключає будь-яку взаємодію з елементами поза ним, поки взаємодія не буде завершена.
- {{CSSxRef(":picture-in-picture")}}
  - : Відповідає елементу, який наразі знаходиться в режимі "картинка в картинці".

## Псевдокласи вводу

Ці псевдокласи дають змогу вибирати формові елементи за їхніми атрибутами й станами до або після взаємодії.

- {{CSSxRef(":autofill")}}
  - : Поле, автоматично заповнене браузером.
- {{CSSxRef(":enabled")}}
  - : Активний елемент інтерфейсу.
- {{CSSxRef(":disabled")}}
  - : Вимкнений елемент інтерфейсу.
- {{CSSxRef(":read-only")}}
  - : Елемент, який користувач не може змінити.
- {{CSSxRef(":read-write")}}
  - : Елемент, який користувач може змінити.
- {{CSSxRef(":placeholder-shown")}}
  - : Елемент, який зараз відображає текст-заповнювач (наприклад, `placeholder`).
- {{CSSxRef(":default")}}
  - : Типовий елемент у наборі.
- {{CSSxRef(":checked")}}
  - : Увімкнений прапорець чи радіокнопка.
- {{CSSxRef(":indeterminate")}}
  - : Елемент у невизначеному стані.
- {{CSSxRef(":blank")}}
  - : Поле, яке є порожнім або містить лише пробіли.
- {{CSSxRef(":valid")}}
  - : Поле з коректним значенням (наприклад, правильна email-адреса).
- {{CSSxRef(":invalid")}}
  - : Поле з некоректним значенням (наприклад, неправильна email-адреса).
- {{CSSxRef(":in-range")}}
  - : Значення поля знаходиться в межах дозволеного діапазону.
- {{CSSxRef(":out-of-range")}}
  - : Значення поля виходить за межі дозволеного діапазону.
- {{CSSxRef(":required")}}
  - : Обов'язкове поле форми.
- {{CSSxRef(":optional")}}
  - : Необов'язкове поле форми.
- {{CSSxRef(":user-valid")}}
  - : Поле, яке стало валідним після взаємодії.
- {{CSSxRef(":user-invalid")}}
  - : Поле, яке стало невалідним після взаємодії.

## Лінгвістичні псевдокласи

Ці псевдокласи відображають мову документа та дозволяють вибирати елементи на основі мови або напрямку тексту.

- {{CSSxRef(":dir", ":dir()")}}
  - : Псевдоклас визначення напрямку вибирає елемент на основі його напрямку, визначеного мовою документа.
- {{CSSxRef(":lang", ":lang()")}}
  - : Вибирає елемент на основі мови його вмісту.

## Псевдокласи розташування

Ці псевдокласи допомагають вибирати посилання та цільові елементи у документі.

- {{CSSxRef(":any-link")}}
  - : Вибирає посилання, що задовольняють умовам {{CSSxRef(":link")}} або {{CSSxRef(":visited")}}.
- {{CSSxRef(":link")}}
  - : Відповідає невідвіданим посиланням.
- {{CSSxRef(":visited")}}
  - : Відповідає відвіданим посиланням.
- {{CSSxRef(":local-link")}}
  - : Вибирає посилання, URL яких збігається з URL поточної сторінки (наприклад, локальні якорі).
- {{CSSxRef(":target")}}
  - : Відповідає елементу, який є ціллю URL.
- {{CSSxRef(":target-within")}}
  - : Відповідає елементу, який сам є ціллю URL або містить таку ціль усередині.
- {{CSSxRef(":scope")}}
  - : Визначає елемент, з якого починається обчислення селекторів.

## Псевдокласи стану ресурсу

Застосовуються до медіа, які можуть відтворюватися (наприклад, відео).

- {{CSSxRef(":playing")}}
  - : Відповідає медіа, що зараз відтворюється.
- {{CSSxRef(":paused")}}
  - : Відповідає медіа, яке поставлене на паузу.

## Псевдокласи часового виміру

Застосовуються, коли відтворюється щось із часовим виміром (наприклад, доріжки [WebVTT](/uk/docs/Web/API/WebVTT_API)).

- {{CSSxRef(":current")}}
  - : Відповідає елементу (або одному з його батьківських елементів), який зараз відображається.
- {{CSSxRef(":past")}}
  - : Відповідає елементу, що повністю передує {{CSSxRef(":current")}}.
- {{CSSxRef(":future")}}
  - : Відповідає елементу, що повністю слідує за {{CSSxRef(":current")}}.

## Псевдокласи деревної структури

Ці псевдокласи дають змогу вибирати елементи за їхнім розташуванням у дереві.

- {{CSSxRef(":root")}}
  - : Кореневий елемент (у HTML зазвичай це `<html>`).
- {{CSSxRef(":empty")}}
  - : Елемент без дочірніх елементів, окрім пробілів.
- {{CSSxRef(":nth-child")}}
  - : Використовує вираз `An+B` для пошуку дочірніх елементів за порядком.
- {{CSSxRef(":nth-last-child")}}
  - : Те саме, що `:nth-child`, але рахує з кінця списку.
- {{CSSxRef(":first-child")}}
  - : Перший дочірній елемент.
- {{CSSxRef(":last-child")}}
  - : Останній дочірній елемент.
- {{CSSxRef(":only-child")}}
  - : Єдиний дочірній елемент (наприклад, один пункт у списку).
- {{CSSxRef(":nth-of-type")}}
  - : Аналог `:nth-child`, але враховує лише елементи певного типу.
- {{CSSxRef(":nth-last-of-type")}}
  - : Аналог `:nth-of-type`, але рахує з кінця списку.
- {{CSSxRef(":first-of-type")}}
  - : Перший дочірній елемент певного типу.
- {{CSSxRef(":last-of-type")}}
  - : Останній дочірній елемент певного типу.
- {{CSSxRef(":only-of-type")}}
  - : Єдиний дочірній елемент вибраного типу.

## Псевдокласи дій користувача

Ці псевдокласи застосовуються, коли користувач взаємодіє з елементом (наприклад, наводить на нього мишу).

- {{CSSxRef(":hover")}}
  - : Збігається з елементом, на який наведено вказівник миші.
- {{CSSxRef(":active")}}
  - : Збігається з елементом у момент натискання чи іншої активації.
- {{CSSxRef(":focus")}}
  - : Збігається з елементом, що перебуває у фокусі.
- {{CSSxRef(":focus-visible")}}
  - : Збігається з елементом у фокусі, який має бути візуально виділений.
- {{CSSxRef(":focus-within")}}
  - : Збігається з елементом, який або сам у фокусі, або містить у собі елемент із фокусом.

## Функціональні псевдокласи

Ці псевдокласи приймають [список селекторів](/uk/docs/Web/CSS/Selector_list) або [нестрогий список селекторів](/uk/docs/Web/CSS/Selector_list#forgiving_selector_list).

- [`:is()`](/uk/docs/Web/CSS/:is)
  - : Збігається з елементами, що підпадають принаймні під один селектор зі списку (список нестрогий).
- [`:not()`](/uk/docs/Web/CSS/:not)
  - : Збігається з елементами, які не підпадають під жоден селектор із аргументу.
- [`:where()`](/uk/docs/Web/CSS/:where)
  - : Збігається з елементами, що підпадають під хоча б один селектор зі списку, **не підвищуючи специфічності**.
- [`:has()`](/uk/docs/Web/CSS/:has)
  - : Збігається з елементом, якщо в ньому (або його нащадках) є інший елемент, що також збігається з указаним селектором.

## Синтаксис

```css
selector:pseudo-class {
  property: value;
}
```

Як і звичайні класи, у селекторі можна використовувати будь-яку кількість псевдокласів.
