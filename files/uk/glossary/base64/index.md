**Base64** (–∑–∞ –æ—Å–Ω–æ–≤–æ—é 64) ‚Äì —Ü–µ –≥—Ä—É–ø–∞ —Å–ø–æ—Ä—ñ–¥–Ω–µ–Ω–∏—Ö –∫–æ–¥—É–≤–∞–Ω—å [–ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –∑ –¥–≤—ñ–π–∫–æ–≤–æ—ó —É —Ç–µ–∫—Å—Ç–æ–≤—É —Ñ–æ—Ä–º—É](https://en.wikipedia.org/wiki/Binary-to-text_encoding), —â–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç—å –¥–≤—ñ–π–∫–æ–≤—ñ –¥–∞–Ω—ñ —É –≤–∏–≥–ª—è–¥—ñ —Ä—è–¥–∫–∞ {{glossary("ASCII")}} –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –ø–µ—Ä–µ–∫–ª–∞–¥—É —ó—Ö —É –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è –∑ —Å–∏—Å—Ç–µ–º–æ—é —á–∏—Å–ª–µ–Ω–Ω—è 64. –¢–µ—Ä–º—ñ–Ω _Base64_ –ø–æ—Ö–æ–¥–∏—Ç—å –≤—ñ–¥ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ [–∫–æ–¥—É–≤–∞–Ω–Ω—è –ø–µ—Ä–µ–¥–∞—á—ñ –≤–º—ñ—Å—Ç—É MIME](https://en.wikipedia.org/wiki/MIME#Content-Transfer-Encoding).

–ö–æ–ª–∏ —Ç–µ—Ä–º—ñ–Ω "Base64" –≤–∂–∏–≤–∞—î—Ç—å—Å—è —Å–∞–º –ø–æ —Å–æ–±—ñ –Ω–∞ –ø–æ–∑–Ω–∞—á–µ–Ω–Ω—è –ø–µ–≤–Ω–æ–≥–æ –∞–ª–≥–æ—Ä–∏—Ç–º—É, –≤—ñ–Ω –∑–∞–∑–≤–∏—á–∞–π –æ–∑–Ω–∞—á–∞—î –≤–µ—Ä—Å—ñ—é Base64, –æ–ø–∏—Å–∞–Ω—É —É [RFC 4648](https://datatracker.ietf.org/doc/html/rfc4648), —Ä–æ–∑–¥—ñ–ª—ñ 4, —è–∫–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î –Ω–∞—Å—Ç—É–ø–Ω–∏–π –∞–ª—Ñ–∞–≤—ñ—Ç –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è —Ä–æ–∑—Ä—è–¥–Ω–∏—Ö —Ü–∏—Ñ—Ä —Å–∏—Å—Ç–µ–º–∏ —á–∏—Å–ª–µ–Ω–Ω—è 64, —Ä–∞–∑–æ–º —ñ–∑ `=` —è–∫ —Å–∏–º–≤–æ–ª–æ–º –≤—ñ–¥—Å—Ç—É–ø—É:

```plain
ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/
```

–ü–æ—à–∏—Ä–µ–Ω–∞ –≤–∞—Ä—ñ–∞—Ü—ñ—è ‚Äì "–±–µ–∑–ø–µ—á–Ω–∏–π –¥–ª—è URL Base64", –∫–æ—Ç—Ä–∏–π –ø—Ä–æ–ø—É—Å–∫–∞—î –≤—ñ–¥—Å—Ç—É–ø —ñ –∑–∞–º—ñ–Ω—é—î `+/` –Ω–∞ `-_`, —â–æ–± —É–Ω–∏–∫–Ω—É—Ç–∏ —Å–∏–º–≤–æ–ª—ñ–≤, —è–∫—ñ –º–æ–∂—É—Ç—å —Å—Ç–≤–æ—Ä—é–≤–∞—Ç–∏ –ø—Ä–æ–±–ª–µ–º–∏ —É —Å–µ–≥–º–µ–Ω—Ç–∞—Ö —à–ª—è—Ö—É URL —ñ –ø–∞—Ä–∞–º–µ—Ç—Ä–∞—Ö –∑–∞–ø–∏—Ç—É.

–°—Ö–µ–º–∏ –∫–æ–¥—É–≤–∞–Ω–Ω—è Base64 –∑–∞–≤–µ–¥–µ–Ω–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –¥–ª—è –∫–æ–¥—É–≤–∞–Ω–Ω—è –¥–≤—ñ–π–∫–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö –¥–ª—è –∑–±–µ—Ä—ñ–≥–∞–Ω–Ω—è —á–∏ –ø–µ—Ä–µ–¥–∞—á—ñ –∑–∞—Å–æ–±–∞–º–∏, —â–æ –º–æ–∂—É—Ç—å –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ –ª–∏—à–µ –∑ —Ç–µ–∫—Å—Ç–æ–º ASCII (–∞–±–æ –ø–µ–≤–Ω–æ—é –Ω–∞–¥–º–Ω–æ–∂–∏–Ω–æ—é ASCII, —â–æ –≤—Å–µ –æ–¥–Ω–æ –Ω–µ –∑–¥–∞—Ç–Ω–∞ –ø—Ä–∏–π–º–∞—Ç–∏ –¥–æ–≤—ñ–ª—å–Ω—ñ –¥–≤—ñ–π–∫–æ–≤—ñ –¥–∞–Ω—ñ). –¶–µ –∑–∞–±–µ–∑–ø–µ—á—É—î, —â–æ –¥–∞–Ω—ñ –∑–∞–ª–∏—à–∞—é—Ç—å—Å—è –Ω–µ–∑–º—ñ–Ω–Ω–∏–º–∏ –ø—ñ–¥ —á–∞—Å —Ç—Ä–∞–Ω—Å–ø–æ—Ä—Ç—É–≤–∞–Ω–Ω—è. –ü–æ—à–∏—Ä–µ–Ω—ñ –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è Base64:

- –ï–ª–µ–∫—Ç—Ä–æ–Ω–Ω–∞ –ø–æ—à—Ç–∞ —á–µ—Ä–µ–∑ [MIME](https://uk.wikipedia.org/wiki/MIME)
- –ó–±–µ—Ä—ñ–≥–∞–Ω–Ω—è —Å–∫–ª–∞–¥–Ω–∏—Ö –¥–∞–Ω–∏—Ö —É [XML](/uk/docs/Web/XML)
- –ö–æ–¥—É–≤–∞–Ω–Ω—è –¥–≤—ñ–π–∫–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö, —â–æ–± —ó—Ö –º–æ–∂–Ω–∞ –±—É–ª–æ –≤–∫–ª—é—á–∏—Ç–∏ –≤ [URL-–∞–¥—Ä–µ—Å—É `data:`](/uk/docs/Web/HTTP/Basics_of_HTTP/Data_URLs)

## –ó–±—ñ–ª—å—à–µ–Ω–Ω—è —Ä–æ–∑–º—ñ—Ä—É –ø—Ä–∏ –∫–æ–¥—É–≤–∞–Ω–Ω—ñ

–ö–æ–∂–Ω–∞ —Ü–∏—Ñ—Ä–∞ Base64 –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—î 6 –±—ñ—Ç –¥–∞–Ω–∏—Ö. –û—Ç–∂–µ, —Ç—Ä–∏ 8-–±—ñ—Ç–æ–≤—ñ –±–∞–π—Ç–∏ –≤–∏—Ö—ñ–¥–Ω–æ–≥–æ —Ä—è–¥–∫–∞ —á–∏ –¥–≤—ñ–π–∫–æ–≤–æ–≥–æ —Ñ–∞–π–ª—É (3√ó8 –±—ñ—Ç = 24 –±—ñ—Ç–∏) –º–æ–∂–Ω–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç–∏ —á–æ—Ç–∏—Ä–º–∞ 6-–±—ñ—Ç–æ–≤–∏–º–∏ —Ü–∏—Ñ—Ä–∞–º–∏ Base64 (4√ó6 = 24 –±—ñ—Ç–∏).

–¶–µ –æ–∑–Ω–∞—á–∞—î, —â–æ –≤–µ—Ä—Å—ñ—è —Ä—è–¥–∫–∞ —á–∏ —Ñ–∞–π–ª—É –≤ Base64 –∑–∞–∑–≤–∏—á–∞–π –ø—Ä–∏–±–ª–∏–∑–Ω–æ –Ω–∞ —Ç—Ä–µ—Ç–∏–Ω—É –±—ñ–ª—å—à–∞ –∑–∞ –≤–∏—Ö—ñ–¥–Ω—ñ –¥–∞–Ω—ñ (—Ç–æ—á–Ω–∏–π –ø—Ä–∏—Ä—ñ—Å—Ç —Ä–æ–∑–º—ñ—Ä—É –∑–∞–ª–µ–∂–∏—Ç—å –≤—ñ–¥ —Ä—ñ–∑–Ω–∏—Ö —Ñ–∞–∫—Ç–æ—Ä—ñ–≤, —Ç–∞–∫–∏—Ö —è–∫ –∞–±—Å–æ–ª—é—Ç–Ω–∞ –¥–æ–≤–∂–∏–Ω–∞ —Ä—è–¥–∫–∞, –æ—Å—Ç–∞—á–∞ –≤—ñ–¥ –¥—ñ–ª–µ–Ω–Ω—è –π–æ–≥–æ –¥–æ–≤–∂–∏–Ω–∏ –Ω–∞ 3 —Ç–∞ —Ç–µ, —á–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è —Å–∏–º–≤–æ–ª–∏ –≤—ñ–¥—Å—Ç—É–ø—É).

## –ü—ñ–¥—Ç—Ä–∏–º–∫–∞ –≤ JavaScript

–î–ª—è —Ä–æ–∑–∫–æ–¥—É–≤–∞–Ω–Ω—è —Ç–∞ –∫–æ–¥—É–≤–∞–Ω–Ω—è —Ä—è–¥–∫—ñ–≤ Base64 –±—Ä–∞—É–∑–µ—Ä–∏ –Ω–∞–¥–∞—é—Ç—å –¥–≤—ñ —Ñ—É–Ω–∫—Ü—ñ—ó JavaScript:

- [`btoa`](/uk/docs/Web/API/btoa) ‚Äì —Å—Ç–≤–æ—Ä—é—î —Ä—è–¥–æ–∫ ASCII, –∑–∞–∫–æ–¥–æ–≤–∞–Ω–∏–π Base64, –∑ —Ä—è–¥–∫–∞ –¥–≤—ñ–π–∫–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö ("btoa" —Å–ª—ñ–¥ —á–∏—Ç–∞—Ç–∏ —è–∫ "binary to ASCII", "–¥–≤—ñ–π–∫–æ–≤–µ –≤ ASCII").
- [`atob`](/uk/docs/Web/API/atob) ‚Äì —Ä–æ–∑–∫–æ–¥–æ–≤—É—î —Ä—è–¥–æ–∫, –∑–∞–∫–æ–¥–æ–≤–∞–Ω–∏–π Base64 ("atob" —Å–ª—ñ–¥ —á–∏—Ç–∞—Ç–∏ —è–∫ "ASCII to binary", "ASCII —É –¥–≤—ñ–π–∫–æ–≤–µ").

> **–ü—Ä–∏–º—ñ—Ç–∫–∞:** Base64 ‚Äì —Ü–µ –¥–≤—ñ–π–∫–æ–≤–µ –∫–æ–¥—É–≤–∞–Ω–Ω—è, –∞ –Ω–µ —Ç–µ–∫—Å—Ç–æ–≤–µ, –∞–ª–µ `btoa` —Ç–∞ `atob` –±—É–ª–∏ –¥–æ–¥–∞–Ω—ñ –¥–æ –≤–µ–±–ø–ª–∞—Ç—Ñ–æ—Ä–º–∏ –¥–æ —Ç–æ–≥–æ, —è–∫ –≤–æ–Ω–∞ –ø–æ—á–∞–ª–∞ –ø—ñ–¥—Ç—Ä–∏–º—É–≤–∞—Ç–∏ –¥–≤—ñ–π–∫–æ–≤—ñ —Ç–∏–ø–∏ –¥–∞–Ω–∏—Ö. –í —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ —Ü—ñ –¥–≤—ñ —Ñ—É–Ω–∫—Ü—ñ—ó –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å —Ä—è–¥–∫–∏ –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è –¥–≤—ñ–π–∫–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö, –ø—Ä–∏ —á–æ–º—É –∫–æ–¥–æ–≤–∞ —Ç–æ—á–∫–∞ –∫–æ–∂–Ω–æ–≥–æ —Å–∏–º–≤–æ–ª—É –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—î –∑–Ω–∞—á–µ–Ω–Ω—è –∫–æ–∂–Ω–æ–≥–æ –±–∞–π—Ç–∞. –¶–µ –ø—Ä–∏–∑–≤–µ–ª–æ –¥–æ –ø–æ—à–∏—Ä–µ–Ω–æ–≥–æ –Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ–≥–æ —Ä–æ–∑—É–º—ñ–Ω–Ω—è, –Ω—ñ–±–∏ `btoa` –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –¥–ª—è –∫–æ–¥—É–≤–∞–Ω–Ω—è –¥–æ–≤—ñ–ª—å–Ω–∏—Ö —Ç–µ–∫—Å—Ç–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö ‚Äì –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è Base64 `data:` URL —Ç–µ–∫—Å—Ç–æ–≤–æ–≥–æ —á–∏ HTML-–¥–æ–∫—É–º–µ–Ω—Ç–∞.
>
> –ü—Ä–æ—Ç–µ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω—ñ—Å—Ç—å –º—ñ–∂ –±–∞–π—Ç–∞–º–∏ —Ç–∞ –∫–æ–¥–æ–≤–∏–º–∏ —Ç–æ—á–∫–∞–º–∏ –Ω–∞–¥—ñ–π–Ω–æ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è –ª–∏—à–µ –¥–ª—è –∫–æ–¥–æ–≤–∏—Ö —Ç–æ—á–æ–∫ –¥–æ `0x7f`. –ö—Ä—ñ–º —Ç–æ–≥–æ, –∫–æ–¥–æ–≤—ñ —Ç–æ—á–∫–∏ –ø–æ–Ω–∞–¥ `0xff` –ø—Ä–∏–∑–≤–µ–¥—É—Ç—å –¥–æ –≤–∏–∫–∏–¥–∞–Ω–Ω—è –ø–æ–º–∏–ª–∫–∏ `btoa`, —á–µ—Ä–µ–∑ –ø–µ—Ä–µ–≤–∏—â–µ–Ω–Ω—è –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è –¥–ª—è 1 –±–∞–π—Ç–∞. –ù–∞—Å—Ç—É–ø–Ω–∏–π —Ä–æ–∑–¥—ñ–ª –¥–µ—Ç–∞–ª—å–Ω–æ –æ–ø–∏—Å—É—î, —è–∫ –æ–±—ñ–π—Ç–∏ —Ü–µ –æ–±–º–µ–∂–µ–Ω–Ω—è –ø—Ä–∏ –∫–æ–¥—É–≤–∞–Ω–Ω—ñ –¥–æ–≤—ñ–ª—å–Ω–æ–≥–æ —Ç–µ–∫—Å—Ç—É Unicode.

## "–ü—Ä–æ–±–ª–µ–º–∞ Unicode"

–û—Å–∫—ñ–ª—å–∫–∏ `btoa` —Ç–ª—É–º–∞—á–∏—Ç—å –∫–æ–¥–æ–≤—ñ —Ç–æ—á–∫–∏ –≤–∏—Ö—ñ–¥–Ω–æ–≥–æ —Ä—è–¥–∫–∞ —è–∫ –∑–Ω–∞—á–µ–Ω–Ω—è –±–∞–π—Ç—ñ–≤, –≤–∏–∫–ª–∏–∫ `btoa` –¥–ª—è —Ä—è–¥–∫–∞ –ø—Ä–∏–∑–≤–µ–¥–µ –¥–æ –≤–∏–∫–∏–¥–∞–Ω–Ω—è –ø–æ–º–∏–ª–∫–∏ "Character Out Of Range" ("—Å–∏–º–≤–æ–ª –ø–æ–∑–∞ –¥–æ–ø—É—Å—Ç–∏–º–∏–º –¥—ñ–∞–ø–∞–∑–æ–Ω–æ–º"), —è–∫—â–æ –∫–æ–¥–æ–≤–∞ —Ç–æ—á–∫–∞ —Å–∏–º–≤–æ–ª—É –ø–µ—Ä–µ–≤–∏—â—É—î `0xff`. –î–ª—è –≤–∏–ø–∞–¥–∫—ñ–≤ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è, –¥–µ –ø–æ—Ç—Ä—ñ–±–Ω–æ –∑–∞–∫–æ–¥—É–≤–∞—Ç–∏ –¥–æ–≤—ñ–ª—å–Ω–∏–π —Ç–µ–∫—Å—Ç Unicode, –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ —Å–ø–æ—á–∞—Ç–∫—É –ø–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ —Ä—è–¥–æ–∫ –Ω–∞ –π–æ–≥–æ —Å–∫–ª–∞–¥–æ–≤—ñ –±–∞–π—Ç–∏ –≤ UTF-8, –∞ –ø–æ—Ç—ñ–º –∑–∞–∫–æ–¥—É–≤–∞—Ç–∏ —Ü—ñ –±–∞–π—Ç–∏.

–ù–∞–π–ø—Ä–æ—Å—Ç—ñ—à–µ —Ä—ñ—à–µ–Ω–Ω—è ‚Äì –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ `TextEncoder` —ñ `TextDecoder` –¥–ª—è –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –º—ñ–∂ UTF-8 —Ç–∞ –æ–¥–Ω–æ–±–∞–π—Ç–æ–≤–∏–º–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è–º–∏ —Ä—è–¥–∫–∞:

```js
function base64ToBytes(base64) {
  const binString = atob(base64);
  return Uint8Array.from(binString, (m) => m.codePointAt(0));
}

function bytesToBase64(bytes) {
  const binString = String.fromCodePoint(...bytes);
  return btoa(binString);
}

// Usage
bytesToBase64(new TextEncoder().encode("a ƒÄ êÄÄ Êñá ü¶Ñ")); // "YSDEgCDwkICAIOaWhyDwn6aE"
new TextDecoder().decode(base64ToBytes("YSDEgCDwkICAIOaWhyDwn6aE")); // "a ƒÄ êÄÄ Êñá ü¶Ñ"
```

## –ü–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –¥–æ–≤—ñ–ª—å–Ω–∏—Ö –¥–≤—ñ–π–∫–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö

–§—É–Ω–∫—Ü—ñ—ó `bytesToBase64` —Ç–∞ `base64ToBytes` –∑ –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ–≥–æ —Ä–æ–∑–¥—ñ–ª—É –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –±–µ–∑–ø–æ—Å–µ—Ä–µ–¥–Ω—å–æ –¥–ª—è –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –º—ñ–∂ —Ä—è–¥–∫–∞–º–∏ Base64 —ñ [`Uint8Array`](/uk/docs/Web/JavaScript/Reference/Global_Objects/Uint8Array).

–Ü–Ω—à–∏–π –≤–∞—Ä—ñ–∞–Ω—Ç: –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–µ –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –º—ñ–∂ URL –∑ –¥–∞–Ω–∏–º–∏ Base64 –º–æ–∂–ª–∏–≤–µ –±–µ–∑–ø–æ—Å–µ—Ä–µ–¥–Ω—å–æ —É –≤–µ–±–ø–ª–∞—Ç—Ñ–æ—Ä–º—ñ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é API [`FileReader`](/uk/docs/Web/API/FileReader) —ñ [`fetch`](/uk/docs/Web/API/Fetch_API):

```js
async function bytesToBase64DataUrl(bytes, type = "application/octet-stream") {
  return await new Promise((resolve, reject) => {
    const reader = Object.assign(new FileReader(), {
      onload: () => resolve(reader.result),
      onerror: () => reject(reader.error),
    });
    reader.readAsDataURL(new File([bytes], "", { type }));
  });
}

async function dataUrlToBytes(dataUrl) {
  const res = await fetch(dataUrl);
  return new Uint8Array(await res.arrayBuffer());
}

// –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è
await bytesToBase64DataUrl(new Uint8Array([0, 1, 2])); // "data:application/octet-stream;base64,AAEC"
await dataUrlToBytes("data:application/octet-stream;base64,AAEC"); // Uint8Array [0, 1, 2]
```
